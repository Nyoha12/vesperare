# 35_FICHE_PERFORMANCE_SUSPENSION_MATIERE_RETOUR

Version : v0.1  
Statut : fiche de performance jouable. Sans outil, sans patch, sans mapping, sans prototype.

## Objet

Cette fiche extrait une troisième situation jouable de :

```text
32_SCENES_PIVOTS_DRAMATURGIES_MUSICALES.md
```

Scène-pivot concernée :

```text
SCENE_PIVOT_SUSPENSION_MATIERE_RETOUR
```

Elle sert à tester musicalement une suspension qui expose la matière sans ambientiser, et qui garde une sortie corporelle possible.

Ce document n’est pas :

```text
un drone ;
un paysage ;
une nappe ;
un interlude ;
une pause ;
un preset de texture ;
un patch ;
un prototype ;
une scène Ableton.
```

C’est une fiche de performance ouverte.

---

# 1. Fonction centrale

Question :

```text
comment suspendre le corps sans le perdre,
exposer une matière sans la rendre décorative,
et revenir ou bifurquer sans casser la forme ?
```

Fonction centrale :

```text
ouvrir un espace d’écoute concentrée,
faire apparaître la matière comme événement musical,
maintenir une trace de corps,
et garder au moins une sortie audible.
```

---

# 2. Sources de départ

## Sources fixes

```text
didgeridoo live ;
guimbardes live.
```

Elles doivent garder une trace vivante même si la scène se retire.

## Sources possibles mais non obligatoires

```text
sub latent ;
gong si présent ;
voix si validée ;
field recording seulement si fonction claire ;
freeze bref avec release ;
loop comme ombre ou trace ;
halo contrôlé.
```

## Sources à éviter au départ

```text
halo large permanent ;
field recording décoratif ;
freeze long ;
sub plein ;
polytexture dense ;
beat complet ;
voix expressive non préparée ;
gong envahissant.
```

---

# 3. Configuration minimale jouable

Version la plus nue :

```text
didgeridoo trace + guimbardes espacées.
```

Version avec sub latent :

```text
didgeridoo trace + guimbardes espacées + sub très bas / discret.
```

Version avec gong :

```text
didgeridoo trace + guimbardes espacées + gong comme champ partiel contrôlé.
```

Version avec voix :

```text
didgeridoo trace + guimbardes espacées + voix souffle / signal / fragment.
```

Version avec freeze :

```text
source claire + freeze bref + release obligatoire + corps restant.
```

Règle :

```text
la suspension n’est valable que si une sortie reste audible.
```

---

# 4. Durée indicative

```text
2–4 minutes : exposition courte ;
5–8 minutes : suspension avec retour ;
9–12 minutes : suspension longue avec bifurcation ou pré-drop depuis le vide.
```

La durée dépend de la qualité d’écoute.

Critère :

```text
la scène dure tant que la matière continue de changer de fonction.
```

Si la matière reste seulement belle, la scène se fige.

---

# 5. Phases de jeu

## Phase 1 — Retrait actif

Objectif : retirer sans arrêter.

Gestes possibles :

```text
réduire ou suspendre la pulsation ;
laisser le didgeridoo devenir souffle / centre grave / trace ;
espacer les guimbardes ;
réduire sub ou le rendre latent ;
fermer l’espace plutôt que l’ouvrir d’emblée ;
laisser apparaître un silence actif.
```

Écoute :

```text
le corps est-il encore senti ?
le retrait crée-t-il de l’attention ?
les sources live restent-elles vivantes ?
le silence est-il chargé ou vide ?
```

Sorties possibles :

```text
exposer une matière ;
revenir au corps ;
armer une tension ;
continuer le retrait ;
préparer un freeze bref.
```

## Phase 2 — Exposition de matière

Objectif : faire entendre une matière comme fonction, pas comme décor.

Gestes possibles :

```text
isoler un formant de guimbarde ;
laisser une attaque métallique respirer ;
faire entendre un souffle de didgeridoo ;
laisser un partiel de gong si présent ;
faire émerger une voix très courte si validée ;
figer très brièvement une source si la release est claire.
```

Écoute :

```text
qu’est-ce qui est exposé ?
pourquoi cette matière mérite-t-elle d’être exposée ?
quelle trace de source reste audible ?
quelle tension ou sortie porte-t-elle ?
```

Sorties possibles :

```text
retrait plus profond ;
freeze court ;
retour au corps ;
pré-drop depuis la matière ;
transfert vers polytexture.
```

## Phase 3 — Suspension tenue

Objectif : tenir l’écoute sans devenir ambient.

Gestes possibles :

```text
maintenir une trace grave ;
faire varier très peu mais précisément ;
laisser les guimbardes ponctuer l’espace ;
ouvrir le halo seulement par nécessité ;
contrôler les queues ;
éviter les couches de confort ;
laisser un objet presque disparaître puis revenir.
```

Écoute :

```text
la suspension a-t-elle une direction ?
le corps est-il latent ou perdu ?
le halo sert-il la forme ?
y a-t-il un risque de paysage ?
la sortie reste-t-elle perceptible ?
```

Sorties possibles :

```text
retour au corps ;
long tension ;
pré-drop ;
withdrawal ;
fermeture ;
transfert vers corps vivant / polytexture.
```

## Phase 4 — Seuil de retour ou de bifurcation

Objectif : rendre le retour possible sans le rendre brutal.

Gestes possibles :

```text
faire revenir une micro-pulsation ;
réintroduire un sub plus clair ;
laisser le didgeridoo redevenir sol ;
resserrer les guimbardes ;
fermer le halo ;
relâcher le freeze ;
utiliser gong ou voix comme fermeture si nécessaire.
```

Écoute :

```text
le retour est-il préparé ?
la suspension accepte-t-elle d’être quittée ?
le corps revient-il comme nécessité ?
la bifurcation est-elle plus forte que le retour ?
```

Sorties possibles :

```text
retour au corps ;
pré-drop depuis le vide ;
long tension ;
corps vivant / polytexture ;
fermeture douce ;
annulation.
```

## Phase 5 — Réincorporation ou disparition

Objectif : sortir avec une trace, pas fermer par habitude.

Gestes possibles :

```text
laisser une guimbarde tenir la dernière trace ;
laisser un souffle didgeridoo ;
stabiliser le sub ;
fermer l’espace ;
stopper le freeze ;
réduire le halo ;
réincorporer une pulsation simple.
```

Écoute :

```text
qu’est-ce qui reste ?
la scène a-t-elle transformé le corps ?
le retour est-il musicalement justifié ?
la disparition est-elle plus forte qu’un retour ?
```

---

# 6. Gestes interdits ou dangereux

```text
faire une nappe permanente ;
ouvrir un halo large sans sortie ;
utiliser field recording comme décor ;
laisser un freeze durer sans release ;
perdre totalement le corps ;
faire revenir le beat brutalement sans préparation ;
chercher la beauté pour elle-même ;
laisser le gong remplir tout l’espace ;
mettre la voix en avant sans fonction ;
transformer la suspension en pause.
```

---

# 7. Points de contrôle pendant le jeu

Toutes les 1–2 minutes, vérifier :

```text
1. Où est la trace de corps ?
2. Quelle matière est exposée ?
3. Quelle est sa fonction ?
4. Quelle sortie est possible ?
5. Le halo sert-il ou remplace-t-il la forme ?
6. Le freeze a-t-il une release ?
7. Les guimbardes restent-elles vivantes ?
8. Le didgeridoo est-il encore agent ?
9. Le field recording, si présent, est-il nécessaire ?
10. Suis-je en train de faire une belle texture ou une scène ?
```

---

# 8. Écoute après performance

Noter rapidement :

```text
durée approximative ;
sources utilisées ;
présence ou absence gong/tambour ;
source exposée ;
type de matière exposée ;
moment où le corps se retire ;
moment où la suspension tient ;
moment où le risque ambient apparaît ;
freeze utilisé ou non ;
release claire ou non ;
retour au corps possible ;
sortie choisie ;
trace finale ;
ce qui a été trop beau ;
ce qui a été trop vide ;
ce qui a gardé la tension.
```

---

# 9. Critères de réussite

La fiche est réussie si :

```text
le corps reste latent ;
la matière exposée a une fonction ;
la suspension a une direction ;
la sortie reste audible ;
le freeze, s’il existe, est relâché ;
le halo ne remplace pas la forme ;
les sources live restent des agents ;
le retour au corps paraît possible sans être obligatoire.
```

---

# 10. Critères d’échec

La fiche échoue si :

```text
la scène devient ambient ;
la matière devient décor ;
la suspension ressemble à une pause ;
le freeze devient drone ;
le field recording remplit sans rôle ;
le retour au corps est impossible ;
le retour au corps est brutal et arbitraire ;
les sources live disparaissent ;
le gong ou le halo prennent tout ;
la beauté coupe la forme.
```

# 11. Variantes à tester

## Variante A — Sans halo

Question :

```text
la suspension peut-elle tenir par trace et silence, sans espace large ?
```

## Variante B — Avec gong partiel

Question :

```text
un partiel de gong peut-il ouvrir sans ambientiser ?
```

## Variante C — Avec freeze très bref

Question :

```text
le freeze intensifie-t-il la matière ou la fige-t-il ?
```

## Variante D — Avec retour au corps lent

Question :

```text
le corps peut-il revenir sans rupture brutale ?
```

## Variante E — Vers pré-drop depuis suspension

Question :

```text
une matière exposée peut-elle devenir seuil de tension ?
```
