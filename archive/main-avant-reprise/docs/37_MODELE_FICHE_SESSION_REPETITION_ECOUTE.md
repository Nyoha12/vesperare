# 37_MODELE_FICHE_SESSION_REPETITION_ECOUTE

Version : v0.1  
Statut : modèle remplissable à copier après chaque répétition. Sans patch, sans mapping, sans prototype.

## Objet

Ce document fournit une fiche de session à copier pour appliquer :

```text
36_PROTOCOLE_JOURNAL_REPETITION_ECOUTE_SCENES_PIVOTS.md
```

Il sert à produire des traces comparables entre les trois fiches jouables :

```text
33_FICHE_PERFORMANCE_CORPS_VIVANT_POLYTEXTURE.md
34_FICHE_PERFORMANCE_TENSION_ARMEE_PRE_DROP.md
35_FICHE_PERFORMANCE_SUSPENSION_MATIERE_RETOUR.md
```

Règle :

```text
remplir vite ;
ne pas chercher l’élégance ;
préserver la mémoire musicale ;
ne pas transformer la fiche en analyse lourde.
```

# Règle de prudence

Ne pas passer au prototype parce qu’une idée est excitante.

Passer à une question technique seulement si :

```text
la fonction existe déjà musicalement ;
elle revient dans plusieurs prises ;
elle est difficile à tenir manuellement ;
elle protège ou prolonge le live ;
elle ne remplace pas le geste ;
son risque principal est identifié ;
sa sortie est claire.
```
