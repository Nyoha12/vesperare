# 34_FICHE_PERFORMANCE_TENSION_ARMEE_PRE_DROP

Version : v0.1  
Statut : fiche de performance jouable. Sans outil, sans patch, sans mapping, sans prototype.

## Objet

Cette fiche extrait une deuxième situation jouable de :

```text
32_SCENES_PIVOTS_DRAMATURGIES_MUSICALES.md
```

Scène-pivot concernée :

```text
SCENE_PIVOT_TENSION_ARMEE_PRE_DROP
```

Elle sert à tester musicalement une tension armée, sans automatisme de drop.

Ce document n’est pas :

```text
une montée standard ;
un build-up ;
un preset de pré-drop ;
un arrangement ;
un patch ;
un prototype ;
une scène Ableton.
```

C’est une fiche de performance ouverte.

---

# 1. Fonction centrale

Question :

```text
comment armer une résolution techno,
sans l’imposer,
sans annoncer mécaniquement le drop,
et sans transformer la tension en effet reconnaissable ?
```

Fonction centrale :

```text
maintenir un seuil actif,
où résoudre, retarder, annuler, bifurquer ou revenir au corps
restent des possibilités réelles.
```

---

# 2. Sources de départ

## Sources fixes

```text
didgeridoo live ;
guimbardes live.
```

Elles doivent rester actives même quand la scène se retire.

## Sources possibles mais non obligatoires

```text
sub retenu ;
gong si présent ;
tambour sur cadre si présent ;
voix si validée ;
repeat court si fonction rythmique claire ;
freeze bref si release prévue.
```

## Sources à éviter au départ

```text
gong spectaculaire immédiat ;
drop complet ;
freeze long ;
voix criée non préparée ;
halo large ;
field recording décoratif ;
repeat démonstratif.
```

---

# 3. Configuration minimale jouable

Version nue :

```text
didgeridoo + guimbardes + retrait partiel du corps.
```

Version avec sub :

```text
didgeridoo + guimbardes + sub retenu.
```

Version avec gong :

```text
didgeridoo + guimbardes + gong armé mais non nécessairement frappé.
```

Version avec tambour :

```text
didgeridoo + guimbardes + tambour comme résolution possible, non comme beat permanent.
```

Version avec voix :

```text
didgeridoo + guimbardes + voix signal seulement si elle a une nécessité formelle.
```

Règle :

```text
plus une source de résolution est forte,
plus elle doit pouvoir être retardée, refusée ou transformée.
```

---

# 4. Durée indicative

```text
2–4 minutes : seuil court, tension claire ;
5–8 minutes : tension étirée avec fausse sortie ;
9–14 minutes : forme longue avec retrait, seuils multiples et bifurcation.
```

La durée dépend de la tension encore disponible.

Critère :

```text
la scène dure tant que l’attente change de qualité.
```

Si l’attente devient simplement attente, elle s’épuise.

---

# 5. Phases de jeu

## Phase 1 — Retrait du corps

Objectif : créer un manque actif, pas un vide décoratif.

Gestes possibles :

```text
retirer une partie de la pulsation ;
retenir le sub ;
laisser le didgeridoo devenir pression incomplète ;
espacer les guimbardes ;
réduire les appuis évidents ;
faire sentir une mesure absente ou différée.
```

Écoute :

```text
le retrait crée-t-il une tension ?
le corps est-il latent mais encore sensible ?
les sources live restent-elles actives ?
le vide appelle-t-il quelque chose ?
```

Sorties possibles :

```text
armer un objet de résolution ;
revenir au corps ;
passer en suspension ;
laisser la tension s’installer plus longtemps.
```

## Phase 2 — Armement

Objectif : rendre une résolution possible sans la déclencher.

Gestes possibles :

```text
sub plus présent mais retenu ;
guimbardes plus nerveuses ou plus insistantes ;
didgeridoo plus tendu, plus serré ;
gong préparé mentalement si présent ;
tambour presque appelé si présent ;
repeat court possible mais non spectaculaire ;
silence court ou trou de densité.
```

Écoute :

```text
sent-on qu’un événement est armé ?
peut-il être retardé ?
peut-il être annulé ?
le corps attend-il sans disparaître ?
```

Sorties possibles :

```text
retarder ;
préparer un faux impact ;
armer un gong ;
introduire une voix signal ;
bifurquer vers suspension.
```

## Phase 3 — Charge sans résolution

Objectif : charger la scène sans obéir à la logique automatique du drop.

Gestes possibles :

```text
augmenter la pression du didgeridoo ;
resserrer les guimbardes ;
laisser le sub pousser puis se retenir ;
créer un repeat puis le raréfier ;
faire croire à une résolution sans la donner ;
laisser le tambour entrer comme menace, pas comme pattern ;
garder le gong hors champ ou en bord de champ.
```

Écoute :

```text
la tension augmente-t-elle ou se répète-t-elle ?
le repeat sert-il le seuil ?
la pression reste-t-elle corporelle ?
la résolution reste-t-elle désirée ?
```

Sorties possibles :

```text
retarder encore ;
fausse résolution ;
résolution sèche ;
annulation ;
suspension ;
retour au corps sans explosion.
```

## Phase 4 — Seuil critique

Objectif : choisir ou refuser une sortie sans casser la forme.

Gestes possibles :

```text
arrêter un repeat ;
retirer sub avant impact ;
laisser la guimbarde seule porter le seuil ;
faire du didgeridoo une trace au lieu d’un moteur ;
gong frappé, effleuré ou totalement retenu ;
tambour comme impact bref ;
freeze très bref si release immédiate prévue.
```

Écoute :

```text
plusieurs issues sont-elles encore vraies ?
le seuil est-il trop attendu ?
la résolution serait-elle trop simple ?
le refus de résolution est-il musical ?
```

Sorties possibles :

```text
résolution ;
fausse résolution ;
annulation ;
bifurcation vers suspension ;
retour au corps ;
long tension.
```

## Phase 5 — Post-trace

Objectif : ne pas effacer ce qui a été construit.

Gestes possibles :

```text
laisser un sub stabilisé ;
laisser une guimbarde comme trace ;
laisser un souffle de didgeridoo ;
contrôler la queue du gong ;
laisser un silence actif ;
réincorporer progressivement le corps ;
fermer le freeze si utilisé.
```

Écoute :

```text
la sortie garde-t-elle la mémoire de la tension ?
la résolution a-t-elle trop tout effacé ?
le corps revient-il comme nécessité ?
la scène peut-elle repartir ailleurs ?
```

---

# 6. Gestes interdits ou dangereux

```text
déclencher un drop par réflexe ;
faire un repeat de démonstration ;
frapper le gong trop tôt ;
faire de la voix un effet ;
laisser le sub devenir flou ;
faire durer un freeze sans release ;
préparer une résolution unique ;
mettre tous les signaux de montée en même temps ;
confondre tension longue et attente statique ;
perdre toute possibilité de retour au corps.
```

---

# 7. Points de contrôle pendant le jeu

Toutes les 1–2 minutes, vérifier :

```text
1. Qu’est-ce qui est armé ?
2. La résolution est-elle nécessaire ou seulement attendue ?
3. Puis-je retarder sans affaiblir ?
4. Puis-je annuler sans faire panne ?
5. Le corps est-il latent ou perdu ?
6. Les guimbardes portent-elles une tension réelle ?
7. Le didgeridoo a-t-il encore une agency ?
8. Le sub pousse-t-il ou brouille-t-il ?
9. Le gong/tambour est-il vraiment disponible ?
10. Quelle sortie reste la plus forte musicalement ?
```

---

# 8. Écoute après performance

Noter rapidement :

```text
durée approximative ;
sources utilisées ;
présence ou absence gong/tambour ;
source qui portait la tension ;
moment où le seuil devient audible ;
moment où la résolution semblait attendue ;
résolution choisie ou refusée ;
fausse résolution éventuelle ;
annulation éventuelle ;
bifurcation éventuelle ;
place du didgeridoo ;
place des guimbardes ;
usage éventuel repeat / loop / freeze ;
qualité du retour au corps ;
ce qui a paru trop attendu ;
ce qui a tenu trop longtemps ;
ce qui mérite d’être gardé.
```

---

# 9. Critères de réussite

La fiche est réussie si :

```text
la tension peut durer sans devenir statique ;
la résolution reste possible mais non obligatoire ;
le corps est retenu mais sensible ;
les sources live restent centrales ;
plusieurs sorties restent crédibles ;
le pré-drop ne devient pas recette ;
la post-trace garde la mémoire de la tension.
```

---

# 10. Critères d’échec

La fiche échoue si :

```text
le drop devient évident ;
le repeat devient effet ;
le gong remplace la tension au lieu de l’accomplir ;
le sub brouille tout ;
la voix apparaît comme gimmick ;
le didgeridoo disparaît comme agent ;
les guimbardes deviennent décor ;
le refus de résolution ressemble à une panne ;
la suspension devient ambient sans décision.
```

# 11. Variantes à tester

## Variante A — Sans résolution

Question :

```text
peut-on tenir une tension complète sans jamais résoudre ?
```

## Variante B — Fausse résolution

Question :

```text
peut-on donner un signe de résolution qui ouvre autre chose ?
```

## Variante C — Gong armé mais absent

Question :

```text
la tension peut-elle fonctionner avec l’idée du gong sans son apparition réelle ?
```

## Variante D — Tambour comme impact sec

Question :

```text
un impact peau bref suffit-il à résoudre sans drop massif ?
```

## Variante E — Freeze court avec release

Question :

```text
un freeze très bref peut-il intensifier le seuil sans devenir drone ?
```
