Version : v0.1  
Statut : notes de recherches stylistiques externes. Support d'analyse, pas source de vérité esthétique.

## Objet

Ce fichier stocke des premiers résultats de recherche sur les influences musicales à réancrer pour Vesperare :

```text
techno ;
dubstep ;
trance ;
acid ;
jazz ;
polytexture ;
texture ;
rythme composite ;
articulation.
```

Usage principal : éviter les caricatures, redonner du contexte musical aux notions internes, et préparer des intégrations ciblées dans les documents source existants.

Règle : commencer par la techno. Le jazz, dubstep, trance, acid et polytexture sont des couches de complexification ou des sources de détails, pas le centre du projet.

# 1. Techno — répétition, réduction, corps, micro-variation

## Source 1

- Titre : Paradygm Shift — Robert Hood
- URL : https://pitchfork.com/reviews/albums/23380-paradygm-shift
- Type : critique musicale.
- Statut : EXTRAIT

## Points extraits

La critique présente Robert Hood comme figure centrale du minimal techno et insiste sur :

```text
boucles arpégées ;
minimalisme ;
efficacité ;
micro-variations ;
changements subtils ;
rythmes imbriqués ;
mouvement perpétuel ;
techno psychédélique au cœur mécanique ;
groove qui semble monter et descendre en même temps.
```

## Apport possible pour Vesperare

À relier à :

```text
répétition non redondante ;
hypnose active ;
pression par continuité ;
variation minimale ;
contour sculpté ;
corps techno sans surcharge.
```

```text
répétition = boucle statique ;
variation = randomisation visible ;
minimalisme = appauvrissement ;
hypnose = stase ;
techno = kick seulement.
```

# 2. Dubstep — attente, sub, syncope, vide, drop non obligatoire

## Source 1

- Titre : Dubstep
- URL : https://en.wikipedia.org/wiki/Dubstep
- Type : synthèse encyclopédique ; utile pour cadrage général.
- Statut : EXTRAIT

## Points extraits

Le dubstep est lié à UK garage, 2-step, drum and bass et à des productions expérimentales. Traits utiles :

```text
rythmes syncopés ;
shuffle / tuplets ;
tempo autour de 132–142 BPM ;
140 comme repère culturel ;
sub-bass proéminent ;
minimalisme ;
silence avant reprise ;
bass drop possible mais non obligatoire ;
structures expérimentales sans drop rigide ;
wobble comme variation rythmique de timbre, volume, filtre ou distorsion.
```

## Apport possible pour Vesperare

À relier à :

```text
tension syncopée ;
pré-drop ;
attente ;
sub comme masse et espace ;
silence relatif ;
résolution rare ;
annulation ;
fausse résolution ;
sonic pressure ;
zone presque drop.
```

## Risque à éviter

```text
dubstep = drop spectaculaire ;
dubstep = wobble cliché ;
pré-drop = recette de build-up ;
sub = effet énorme ;
repeat = stutter démonstratif.
```

# 5. Jazz — comping, voicings, extensions, placement, écoute

## Source 1

- Titre : Comping (jazz)
- URL : https://en.wikipedia.org/wiki/Comping_%28jazz%29
- Type : synthèse encyclopédique ; utile pour vocabulaire.
- Statut : EXTRAIT

## Points extraits

Le comping combine :

```text
accords ;
rythmes ;
contre-mélodies ;
soutien d'un soliste ;
interaction ;
adaptation au musicien principal ;
écoute ;
voicings simples ou étendus ;
extensions : 9ths, 13ths ;
altérations ;
réharmonisation ;
textures pouvant devenir trop chargées si plusieurs instruments harmoniques agissent sans écoute.
```

## Apport possible pour Vesperare

Le jazz peut aider à penser l'accompagnement comme :

```text
soutien réactif ;
écoute du soliste / source live ;
champ harmonique ;
voicing distribué ;
interaction fine ;
contre-mouvement ;
non-remplissage ;
place laissée à la source principale.
```

## Risque à éviter

```text
jazz = accords riches ;
voicing = décoration harmonique ;
comping = remplissage ;
humanize = swing ;
harmonie = couleur autonome ;
plus de notes = plus de finesse.
```

## Source 3

- Titre : Altered chord
- URL : https://en.wikipedia.org/wiki/Altered_chord
- Type : synthèse encyclopédique ; utile pour vocabulaire.
- Statut : EXTRAIT

## Points extraits

Un altered chord modifie un ou plusieurs tons diatoniques par des hauteurs chromatiques voisines. En jazz, l'altération peut ajouter des notes hors gamme, enrichir une progression, créer des tensions vers des voisins diatoniques.

## Apport possible pour Vesperare

À traduire comme :

```text
tension non tempérée ;
altération de partiels ;
voix instable ;
champ de friction ;
conduite vers résolution ou non-résolution ;
couleur liée à fonction, pas effet.
```

## Risque à éviter

```text
jazzifier explicitement ;
plaquer de l'harmonie ;
forcer le tempérament égal ;
perdre les tensions instrumentales naturelles.
```

## Source 4

- Titre : Does it Swing? Microtiming Deviations and Swing Feeling in Jazz
- URL : https://arxiv.org/abs/1904.03442
- Type : recherche empirique.
- Statut : À COMPARER

## Points extraits provisoires

L'article questionne le lien direct entre microtiming deviations et sensation de swing. Le résultat rapporté dans le résumé suggère que les micro-écarts naturels ne sont pas forcément un facteur essentiel du swing ressenti dans toutes les conditions expérimentales.

## Apport possible pour Vesperare

Attention : ne pas traduire jazz/finesse rythmique par simple humanize ou microtiming aléatoire.

À relier plutôt à :

```text
placement ;
accentuation ;
relation entre parties ;
écoute ;
comping ;
fonction rythmique ;
corps global.
```

## Risque à éviter

```text
swing = retard aléatoire ;
finesse rythmique = humanize ;
jazz feel = microtiming automatique.
```

# 6. Texture / polytexture / articulation

## Source 1

- Titre : Texture (music)
- URL : https://en.wikipedia.org/wiki/Texture_%28music%29
- Type : synthèse théorique générale.
- Statut : EXTRAIT

## Points extraits

La texture musicale concerne la manière dont les matériaux mélodiques, rythmiques et harmoniques se combinent. Elle peut être décrite selon :

```text
densité ;
épaisseur ;
ambitus / largeur ;
nombre de voix ;
relations entre voix ;
rôles : primary melody, secondary melody, static support, harmonic support, rhythmic support, etc. ;
polyphonie ;
hétérophonie ;
homophonie ;
textures composées ou mixtes.
```

## Apport possible pour Vesperare

À relier à :

```text
polytexture percussive raffinée ;
densité articulée ;
rôles d'objets ;
relations entre couches ;
lisibilité ;
masquage ;
harmonie de timbres ;
scènes polytexture body.
```

## Risque à éviter

```text
polytexture = beaucoup de couches ;
densité = remplissage ;
texture = ambiance ;
relation entre couches non définie ;
masquage du live.
```

## Source 2

- Titre : Rhythm — Composite rhythm
- URL : https://en.wikipedia.org/wiki/Rhythm
- Type : synthèse théorique générale.
- Statut : EXTRAIT

## Points extraits

Le composite rhythm ou attack-point rhythm est le rythme résultant de l'ensemble des articulations entre toutes les voix d'une texture.

## Apport possible pour Vesperare

À relier à :

```text
polytexture ;
interlocking ;
contrepoint percussif ;
rythme global résultant ;
relation guimbardes / percussions / repeats / objets courts ;
lisibilité corporelle de la densité.
```

## Risque à éviter

```text
penser chaque couche séparément ;
perdre le rythme résultant ;
confondre densité et propulsion ;
masquer la grille corporelle.
```

## Source 3

- Titre : Articulation (music)
- URL : https://en.wikipedia.org/wiki/Articulation_%28music%29
- Type : synthèse théorique générale.
- Statut : EXTRAIT

## Points extraits

L'articulation détermine comment un événement sonore commence, se termine, sa longueur, son attaque, son decay, son timbre, sa dynamique et parfois sa hauteur.

## Apport possible pour Vesperare

À relier à :

```text
contour sculpté ;
netteté sensible ;
brillance rythmique ;
attaque guimbarde ;
objets courts ;
impact-fondation ;
polytexture lisible ;
contrôle de contour.
```

## Risque à éviter

```text
contour = enveloppe technique seulement ;
articulation = détail secondaire ;
attaque = agressivité ;
brillance = aigu décoratif.
```
