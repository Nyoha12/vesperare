# 01_CAHIER_DES_PHENOMENES_SONORES

Version : v0.3  
Statut : cahier phénoménologique de transfert entre style, objets sonores, trajectoires et spécification technique.

## Objet

Ce document définit les phénomènes sonores centraux de Vesperare. Il sert de pont entre :

```text
05_CARTOGRAPHIE_STYLE_PERSONNEL_TECHNO
06_CARTOGRAPHIE_INFLUENCES_STYLE_VESPERARE
07_PITCH_HARMONIQUES_MICROTONALITE
08_PERFORMANCE_SCENES_TRAJECTOIRES_OBJETS
```

et :

```text
02_SPECIFICATION_MAX_FOR_LIVE
```

Il ne décrit donc pas seulement des sons désirables. Il transforme les décisions esthétiques, stylistiques et performatives en phénomènes manipulables par le futur système.

## Principe général

Vesperare est une techno personnelle naturalisée : les fonctions techno sont conservées par leurs effets perceptifs, corporels, rythmiques et formels, mais elles peuvent être portées par des matières réelles, acoustiques, instrumentales, vocales, naturelles, microtonales, hybrides ou synthétisées comme phénomènes physiques.

On ne demande pas seulement :

```text
ce son est-il techno ?
```

mais :

```text
quelle fonction peut-il porter ?
quel phénomène perceptif produit-il ?
quel objet sonore ou groupe d’objets l’incarne ?
quel rôle joue-t-il dans la scène ?
à quelles trajectoires peut-il participer ?
quelles dimensions affecte-t-il ?
qu’ajoute sa matière ?
qu’est-ce qu’il risque de faire perdre ?
```

## Niveaux à distinguer

Pour éviter les confusions, chaque phénomène doit être pensé selon plusieurs niveaux.

```text
fonction musicale : ce que l’élément fait ;
phénomène sonore : comment cette fonction devient perceptible ;
objet sonore : source ou groupe de sources qui porte le phénomène ;
rôle musical : fonction momentanée de l’objet dans une scène ;
trajectoire : transformation à laquelle le phénomène peut participer ;
dimension affectée : rythme, spectre, espace, harmonie, corps, lisibilité, etc. ;
variable interne : représentation possible dans le système Max for Live.
```

Un même objet peut porter plusieurs phénomènes. Un même phénomène peut être porté par plusieurs objets. Une même trajectoire peut distribuer un phénomène entre plusieurs objets.

# 1. Impact-fondation corporel

## Définition

Phénomène par lequel un son réinstalle le corps de la musique : sol, pulsation, impact, autorité temporelle, pression physique.

Il correspond à une fonction souvent portée par le kick techno, mais il ne doit pas nécessairement sonner comme un kick électronique.

## Fonctions musicales

```text
ancrer ;
frapper ;
réincorporer ;
stabiliser ;
marquer une résolution ;
préserver le corps techno.
```

## Objets sonores possibles

```text
tambour profond ;
peau grave ;
grosse percussion traitée ;
impact acoustique ;
objet frappé ;
gong sculpté ;
didgeridoo percussif ou filtré ;
résonance grave resserrée ;
sub stable ;
son naturel grave transformé ;
synthèse à comportement acoustique.
```

## Rôles possibles

```text
fondation principale ;
retour au corps ;
masse de résolution ;
point de réalignement ;
trace corporelle dans une suspension ;
contrepoids à une beauté autonome.
```

## Trajectoires associées

```text

pré-drop → résolution ;
suspension → retour fondation ;
matière exposée → techno naturalisée ;
peak corporel non agressif.
```

## Dimensions affectées

```text

rythme ;
sub ;
pression ;
forme ;
lisibilité ;

```

## Risques

```text

irrégularité excessive → perte d’ancrage ;
```

# 2. Pression spectrale corporelle

## Définition

Sensation que le son occupe physiquement et mentalement l’espace d’écoute. Elle ne dépend pas seulement du volume, mais de la qualité du grave, du bas-médium, de la densité, de la stabilité, du contour et du confort spectral.

## Fonction dans le projet

Garder l’intensité techno et hardcore sans utiliser principalement l’agression des aigus, le volume excessif ou la dureté punitive.

# 3. Contour sculpté / netteté sensible

## Définition

Remplacement de l’idée trop froide de sécheresse. Le projet ne cherche pas nécessairement une techno froide, sèche ou aride. Ce qui est retenu : son dessiné, attaque lisible, queue maîtrisée, matière bien formée, enveloppe raffinée, son fini et bien senti.

## Critère

Le critère n’est pas :

```text
est-ce sec ?
```

mais :

```text
est-ce que le son tient sa fonction sans se dissoudre ?
est-ce que son attaque et sa queue sont maîtrisées ?
est-ce que sa richesse reste lisible ?
est-ce que l’objet garde sa personnalité sans perdre son rôle ?
```


## Rôles possibles

```text
articulation ;

saillance ;

limite de halo ;
signal rythmique.
```

## Trajectoires associées

```text

préparation de résolution.
```

## Dimensions affectées

# 15. Relation avec les phénomènes plus anciens

Les notions de densité, grain, halo, saillance, équilibre spectral, partiels, masquage, glissement d’appui et interaction live restent pertinentes. Elles doivent maintenant être relues à travers le projet actuel.

Exemples :

```text
densité → pression corporelle + lisibilité techno + polytexture ;
grain → contour sculpté + matière acoustique + torsion ;
halo → espace musical + suspension + risque d’ambientisation ;
saillance → impact-fondation + voix + événement formel ;
spectre → confort, pression, intensité sans agression ;
polytexture → enrichissement techno, pas importation de style ;
partiels → harmonie, microtonalité, objets et trajectoires ;
interaction live → objet sonore multidimensionnel, pas rôle fixe.
```
