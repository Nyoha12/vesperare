Projet : Vesperare

Cette fiche extrait les besoins stylistiques. Elle ne definit pas encore une architecture technique, une interface, des pistes Ableton, des devices ou des mappings.

Regle generale :

Chaque element stylistique doit pouvoir etre present, absent, latent, ponctuel, frontal ou structurel selon la scene. Aucun element n'est permanent par principe.

## BS-01 - Techno comme socle recuperable

Statut : confirme.

Contexte d'inspiration :

- techno fondatrice ;
- Jeff Mills pour hypnose, aspiration, force du cadre ;
- Robert Hood pour temps long, repetition, detail rythmique ;
- club, nuit, pression corporelle, intensite.

Besoin :

Vesperare doit garder la techno comme socle stylistique principal, mais ce socle n'a pas besoin d'etre present en surface en permanence.

Signes auditifs possibles :

- cadre rythmique perceptible ;
- repetition hypnotique ;
- retour de motif ;
- pression corporelle ;
- tension longue ;
- simplicite apparente mais active ;
- energie nocturne ;
- retour du corps apres suspension.

Regles :

- la techno peut etre explicite ;
- elle peut etre reincarnee par des supports acoustiques ;
- elle peut etre latente ;
- elle peut etre suspendue tres longtemps ;
- elle doit pouvoir etre recuperee ou reinstallee.

Risques :

- techno standard ;
- dark techno cliche ;
- boucle morte ;
- perte totale du corps ;
- suspension sans memoire.

Garde-fous :

- conserver au moins une pression, une memoire ou une direction ;
- distinguer fonction techno et timbre techno ;
- eviter les sons electroniques reconnaissables comme identite dominante.

## BS-02 - Techno reincarnee, pas techno decoree

Statut : confirme.

Besoin :

Les fonctions techno doivent pouvoir etre portees par des supports acoustiques, naturels, vocaux, instrumentaux ou pseudo-acoustiques plausibles.

Fonctions a reincarner :

- impact-fondation ;
- micro-pulsation ;
- pression grave ;
- masse ;
- halo ;
- tension ;
- retour de cadre ;
- build ;
- resolution ;
- densite d'attaques.

Supports possibles :

- didgeridoo ;
- gong ;
- tambour ;
- voix ;
- peaux ;
- bois ;
- metaux ;
- souffles ;
- frottements ;
- insectes ou oiseaux si leur precision sert une fonction ;
- resonances, cavites, partiels.

Risques :

- sons naturels poses sur une techno standard ;
- decor naturaliste ;
- folklore involontaire ;
- perte de la fonction techno initiale.

Garde-fous :

- nommer la fonction avant la source ;
- verifier attaque, lisibilite temporelle, propulsion, controle spectral ;
- refuser tout support qui ne fait qu'illustrer.

## BS-03 - Plausibilite acoustique et ambiguite

Statut : confirme.

Besoin :

Les sons electroniques reconnaissables doivent etre evites. Les sons peuvent etre ambigus, et meme impossibles au sens strict, mais ils doivent rester plausibles comme extensions acoustiques.

Signes de plausibilite :

- cause possible : frappe, souffle, friction, pression, cavite, membrane, metal, voix, air ;
- morphologie credible : attaque, corps, sustain, queue, silence ;
- evolution spectrale organique : partiels, formants, battements, rugosite, instabilite ;
- espace plausible : distance, profondeur, surface, reflexion ;
- energie corporelle lisible : geste, effort, resistance, relachement.

Ce qui est accepte :

- acoustique direct ;
- acoustique etendu ;
- ambigu plausible ;
- hybride fragile si la logique acoustique domine.

Ce qui est a eviter :

- preset reconnaissable ;
- synth lead identifiable ;
- supersaw ;
- pluck trance ;
- wobble bass ;
- riser EDM ;
- auto-tune reconnaissable ;
- sidechain pompe ;
- glitch decoratif.

Formule :

Un son peut etre impossible, mais il doit laisser imaginer une cause physique.

## BS-04 - Pression rythmique prioritaire

Statut : confirme.

Besoin :

La pression rythmique est une force stylistique prioritaire de Vesperare.

Elle ne se limite pas a une pulsation explicite.

Formes possibles :

- pulse explicite ;
- pulse implicite ;
- interlocking ;
- syncope de masse ;
- accent fantome ;
- respiration de matiere ;
- densite d'attaques ;
- dissonance metrique ;
- rythme spatial ;
- retrait rythmique ;
- retour de cadre ;
- rythme instrumental live.

Signes de reussite :

- le corps reste oriente ;
- la tension est lisible ;
- la complexite donne une direction ;
- le retrait cree une attente ;
- le retour a une valeur musicale.

Risques :

- complexite demonstrative ;
- pseudo-humanisation aleatoire ;
- grille perdue sans intention ;
- pulsation trop standard ;
- variation qui detruit l'hypnose.

Garde-fous :

- une couche directrice doit pouvoir etre identifiee ;
- la polytexture doit servir le corps ou l'attente ;
- la variation doit avoir une cible musicale.

## BS-05 - Espace comme pression active

Statut : confirme.

Besoin :

L'espace n'est pas un decor. Il doit pouvoir devenir pression, profondeur, vertige, attente, confort, separation ou retour.

Fonctions de l'espace :

- profondeur ;
- distance ;
- proximite ;
- scene ;
- vertige ;
- separation des plans ;
- suspension ;
- retour ;
- confort sound system ;
- pression par vide.

Risques :

- ambient decoratif ;
- grande reverb partout ;
- flou du groove ;
- halo qui mange les attaques ;
- espace spectaculaire sans fonction.

Garde-fous :

- tout espace doit avoir une fonction ;
- le halo doit etre dose selon densite, grave, live et attaque ;
- espace techno ne signifie pas ambient ;
- le vide doit garder tension ou direction.

## BS-06 - Halo controle

Statut : confirme.

Besoin :

Le halo doit etre une fonction musicale controlee, pas un effet permanent.

Fonctions possibles :

- prolonger un impact ;
- lier des evenements ;
- creer une memoire ;
- soutenir une suspension ;
- porter des partiels ;
- ouvrir l'espace ;
- creer une tension harmonique ;
- transformer une attaque en champ.

Risques :

- halo permanent ;
- queue longue non justifiee ;
- attaque effacee ;
- grave envahi ;
- texture devenue nappe ;
- perte du groove.

Garde-fous :

- limiter les queues longues ;
- reduire le halo quand la densite monte ;
- reduire le halo generatif quand le live a deja du sustain ;
- deplacer ou couper le halo quand une attaque porte la fonction principale ;
- eviter le halo long dans le grave sans raison forte.

## BS-07 - Techno suspendue longue

Statut : confirme.

Besoin :

Une section peut suspendre la techno tres longtemps.

Conditions :

- garder une pression ;
- garder une memoire ;
- garder une direction ;
- ou assumer une absence tres claire comme tension formelle.

Types de suspension :

- pulse fantome ;
- suspension spatiale ;
- suspension spectrale ;
- suspension instrumentale ;
- pre-retour ;
- suspension rituelle ;
- faux retour.

Strategie de retour :

- cristallisation d'un accent ;
- retour du grave ;
- apparition d'une pulsation deja cachee ;
- contraction de l'espace ;
- densification d'attaques ;
- alignement polytextural ;
- resolution microtonale ;
- impact rare.

Risques :

- ambient decoratif ;
- perte d'attention ;
- retour arbitraire ;
- absence de forme.

## BS-08 - Polytexture rythmico-spectrale

Statut : confirme.

Contexte d'inspiration :

Keita Ogawa comme point de depart phenomenologique pour tissage, interlocking, attaques, plans, contrepoints, richesse timbrale.

Besoin :

La polytexture doit etre disponible comme facette operatoire, non permanente.

Elle doit etre un entre-deux :

- rythmique ;
- percussif ;
- spectral ;
- harmonique ;
- vocal ;
- microtonal ;
- spatial.

Fonctions :

- densifier le groove ;
- remplacer les hats ;
- creer un contrepoint techno ;
- enrichir la finesse rythmique ;
- creer un reseau d'harmonie de timbres ;
- servir de transition ;
- devenir texture principale ponctuelle.

Risques :

- virtuosite decorative ;
- surcharge medium ;
- folklore ;
- complexite abstraite ;
- perte du corps.

Garde-fous :

- chaque couche doit avoir un role ;
- limiter les couches actives ;
- garder une hierarchie perceptive ;
- preserver le grave et le live ;
- pouvoir retirer la polytexture.

## BS-09 - Melodie variable et naturalisee

Statut : confirme.

Besoin :

Vesperare n'est pas anti-melodique. La melodie peut exister sous plusieurs regimes, mais elle doit rester liee a la matiere, au corps, au souffle, au grave, a la voix, aux partiels ou a la scene.

Regimes autorises :

- melodie absente ;
- centre ressenti ;
- accent-melody ;
- basse-motif ;
- melodie de timbre ;
- melodie de partiels ;
- ligne acid naturalisee ;
- fragment vocal ;
- hook minimal ;
- melodie microtonale.

Risques :

- lead banal ;
- hook generique ;
- melodic techno cliche ;
- vocal hook pop ;
- ligne acid autonome ;
- melodie jolie mais sans fonction.

Garde-fous :

- toute melodie doit avoir une source ou fonction ;
- privilegier timbre, formants, partiels, accents, grave, voix ;
- permettre apparition, disparition et transformation ;
- eviter les progressions emotionnelles generiques.

## BS-10 - Didgeridoo libre et accommodable

Statut : confirme.

Besoin :

Le didgeridoo doit pouvoir rester libre. Il ne doit pas etre force a devenir basse, drone, input, controleur ou role fixe.

Libertes a proteger :

- ne pas piloter le systeme ;
- ne pas jouer la basse ;
- ne pas etre traite ;
- ne pas rester en drone ;
- pouvoir devenir central ponctuellement ;
- pouvoir rester presence variable.

Modes stylistiques possibles :

- libre ;
- protege ;
- basse adaptee ;
- grave partage ;
- guide harmonique ;
- source traitee ;
- fonctionnalise ;
- perturbateur ;
- hybride.

Risques :

- didgeridoo masque ;
- didgeridoo reduit a basse ;
- connotation ethno ;
- traitement gadget ;
- protection excessive qui fait perdre la pression.

Garde-fous :

- le contexte peut s'adapter pour qu'il reste perceptible ;
- le grave genere doit pouvoir se retirer, alterner ou se deplacer ;
- le didgeridoo doit garder son autonomie expressive.

## BS-11 - Voix comme presence, matiere, signal

Statut : confirme.

Besoin :

La voix peut etre instrument, effet, matiere, signal, presence, cri, souffle, fragment ou parole non necessairement chantee.

Fonctions :

- signal formel ;
- presence humaine ;
- matiere de souffle ;
- rythme consonantique ;
- hook minimal ;
- texture ;
- transgression ;
- harmonie par formants.

Ce qui n'est pas obligatoire :

- chanson ;
- lead vocal ;
- narration centrale ;
- texte intelligible ;
- vocal trance.

Risques :

- gimmick vocal ;
- melodrame ;
- hook pop ;
- auto-tune reconnaissable ;
- parole qui prend toute la scene.

Garde-fous :

- definir le role de la voix ;
- parametrer intelligibilite ;
- garder la voix compatible avec matiere, espace ou rythme ;
- permettre voix brute ou nue seulement si la forme le justifie.

## BS-12 - Microtonalite, partiels, pitch large

Statut : confirme.

Besoin :

La microtonalite fait partie de Vesperare, mais pas comme demonstration d'accordage.

Elle sert a organiser :

- partiels ;
- harmoniques ;
- hauteurs ressenties ;
- centres spectraux ;
- formants ;
- battements ;
- tensions ;
- resolutions ;
- relation sub/halo ;
- gongs, bols, voix, didgeridoo.

Regimes :

- par defaut : couleur, tension subtile, battements, materialite ;
- ponctuel : microtonalite perceptible et structurante.

Risques :

- theorie audible pour elle-meme ;
- gamme exotique plaquee ;
- perte du corps ;
- abstraction froide ;
- outil ingerable.

Garde-fous :

- relier microtonalite a tension, resolution, naturalisation ou expressivite ;
- privilegier sources acoustiques comme guides ;
- distinguer couleur subtile et moment structurel.

## BS-13 - Dubstep comme attente, syncope, grave, resolution

Statut : confirme.

Besoin :

Le dubstep doit etre retenu comme source de phenomenes, pas comme style entier.

Phenomenes retenus :

- syncope ;
- attente ;
- drop rare ;
- resolution rythmique ;
- sub comme fonction corporelle ;
- masse grave ;
- voix criee ;
- gong/rugissement ;
- microtonalite grave.

Risques :

- wobble bass ;
- brostep ;
- drop automatique ;
- basse electronique reconnaissable ;
- violence sonore gratuite.

Garde-fous :

- le drop est rare, prepare, jamais automatique par defaut ;
- la resolution peut etre rythmique, spectrale, corporelle, harmonique ou timbrale ;
- le sub est une fonction de grave, pas un timbre impose.

## BS-14 - Dub / sound system comme profondeur et confort

Statut : confirme.

Besoin :

Le sound system interesse comme science de l'espace, du grave, du confort et de la profondeur.

Fonctions :

- sub confortable ;
- pression par vide ;
- profondeur ;
- separation des plans ;
- flow ;
- intensite sans agression.

Risques :

- devenir dub/reggae ;
- delay cliche ;
- basse dub typique ;
- confort qui affadit ;
- espace decoratif.

Garde-fous :

- espace et grave doivent avoir une fonction ;
- garder la pression corporelle ;
- eviter les marqueurs de style dub/reggae.

## BS-15 - Jazz comme champ de tension et finesse

Statut : confirme.

Besoin :

Le jazz doit etre traduit en champs de tension, couleurs harmoniques, voicings, finesse rythmique et relation harmonie/rythme, sans importer le style jazz par defaut.

Ce qui est retenu :

- harmonie comme matiere ;
- voicings distribues ;
- tensions non resolues ;
- cadence evitee ;
- accentuation fine ;
- ponctuation vivante ;
- micro-instabilite.

Risques :

- swing cliche ;
- accords jazz plaques ;
- walking bass ;
- humanize aleatoire ;
- intellectualisation.

Garde-fous :

- harmonie comme matiere d'abord ;
- ecriture harmonique seulement ponctuelle ;
- toute tension doit servir pression, espace, forme ou retour.

## BS-16 - Trance comme suspension et expansion naturalisees

Statut : confirme.

Besoin :

La trance interesse par suspension, ouverture, expansion, build-up, breakdown et peak sans brutalite.

Procedures a naturaliser :

- supersaw -> essaim harmonique acoustique ;
- pluck -> clochettes, bois, gouttes, percussions tonales ;
- arpege -> partiels, cloches, voix decoupee ;
- pad -> halo de gongs, voix, bols ;
- gated pad -> halo coupe rythmiquement ;
- riser -> souffle, friction, gong frotte ;
- snare roll -> peaux, frottements, polytexture resserree ;
- sidechain -> respiration de matiere ;
- vocal trance -> voix suspendue, fragment, voyelle.

Risques :

- euphorie generique ;
- brillance trop electronique ;
- peak EDM ;
- halo decoratif ;
- vocal trance.

Garde-fous :

- naturaliser chaque procede ;
- garder une pression active ;
- peak = matiere, espace, corps ou resolution, pas forcement explosion.

## BS-17 - Acid comme ligne vivante et torsion

Statut : confirme.

Besoin :

L'acid doit etre retenu comme source de ligne vivante, resonance controlee, torsion et hypnose par variation de timbre, pas comme reproduction TB-303.

Phenomenes retenus :

- glissement de resonance ;
- glissement de formant ;
- gorge qui se serre ;
- metal qui grince ;
- bol qui se tord ;
- didgeridoo qui module ;
- friction ;
- pression de matiere ;
- reverb filtree qui parle.

Risques :

- squelch acid reconnaissable ;
- pattern 303 ;
- module acid autonome ;
- filtre gadget.

Garde-fous :

- timbre avant notes ;
- ligne comme matiere ;
- coherence avec scene active ;
- jamais bloc acid separe par defaut.

## BS-18 - Intensite phenomenale sans agression

Statut : confirme.

Besoin :

Vesperare doit pouvoir atteindre une intensite enorme, parfois transgressive, sans fonder cette intensite sur l'agression sonore gratuite.

Sources d'intensite :

- pression rythmique ;
- pression grave ;
- pression spectrale ;
- espace immense ;
- rarete ;
- retour de cadre ;
- rugissement ;
- cri ;
- saturation organique ;
- microtonalite exposee ;
- contraste nudite/masse.

Risques :

- volume comme solution ;
- aigus punitifs ;
- saturation dure sans fonction ;
- transgression permanente ;
- fatigue ;
- violence sans consequence formelle.

Garde-fous :

- identifier la source de l'intensite ;
- prevoir une sortie ;
- distinguer intensite, transgression, pic et agression ;
- garder le confort comme contrainte musicale.

## BS-19 - Beaute fonctionnelle et beaute autonome

Statut : confirme.

Besoin :

La beaute peut etre fonctionnelle ou presque autonome.

Beaute fonctionnelle :

- un son est beau parce qu'il est exactement a sa place ;
- bonne intensite ;
- bon contour ;
- bonne duree ;
- bonne relation au groove, a l'espace et au silence.

Beaute autonome :

- moment de matiere ;
- gong expose ;
- voix nue ;
- champ de partiels ;
- suspension ;
- halo ;
- microtonalite perceptible.

Risques :

- beaute decorative ;
- moment pur sans forme ;
- mystere devenu flou ;
- transport devenu euphorie generique.

Garde-fous :

- tout moment autonome doit garder pression, attente, transition, exposition, transgression, memoire ou direction.

## BS-20 - Scene auditive et hierarchie perceptive

Statut : confirme.

Besoin :

Vesperare doit etre pense comme scene auditive : plans, fusion, separation, masquage, profondeur, proximite, stabilite et emergence d'objets.

Axes :

- objet sonore ;
- texture sonore ;
- entre-deux objet/texture ;
- avant-plan ;
- fond ;
- espace ;
- grave ;
- plan live.

Risques :

- tout en objets isoles ;
- tout en texture ;
- densite illisible ;
- fond qui mange l'avant-plan ;
- objet principal sans contour.

Garde-fous :

- garder une hierarchie perceptive ;
- definir ce qui ressort, fusionne, se separe ;
- verifier les monopoles perceptifs : sub, articulation, brillant continu, halo long, transgression.

## BS-21 - Anti-collage stylistique

Statut : confirme.

Besoin :

Les inspirations peuvent etre superposees ou absentes, mais elles ne doivent pas devenir des calques stylistiques.

Regle :

On ne demande pas "est-ce que ca sonne dubstep/jazz/trance/acid ?", mais :

- quelle fonction est active ;
- par quel support ;
- avec quelle pression ;
- avec quelle plausibilite acoustique ;
- avec quel risque ;
- avec quel garde-fou.

Glissements a eviter :

- techno organique -> collection d'effets ;
- didgeridoo -> source sub ;
- hypnose -> randomisation ;
- complexite -> accumulation ;
- naturalisation -> decor ;
- melodie -> lead banal ;
- espace -> ambient.

Garde-fou final :

Toute inspiration doit etre traduite en phenomene musical situe, sinon elle reste en hypothese.
