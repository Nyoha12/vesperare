Projet : Vesperare
Statut : fiche synchronisee avec reverb/espace v0.2 et generation v0.4
Date : 2026-06-21

Sources :

Cette fiche decrit les besoins stylistiques. Elle ne definit pas encore l'architecture technique, l'interface, les mappings, les pistes ou les devices.

Cette v0.4 ne reouvre pas tous les reservoirs stylistiques.

Elle synchronise seulement les garde-fous stylistiques avec les decisions recentes sur :

- espaces longs decides ;
- halo Lock / Force / Cut / Auto ;
- grave spatial et sub ;
- didgeridoo libre et accommodable ;
- naturalisation non decorative ;
- distinction entre ambient subi et espace choisi.

Regle generale :

Chaque element peut etre absent, latent, ponctuel, frontal ou structurel. Son activation doit toujours etre justifiee par une fonction musicale situee.

Regle de non-automatisme :

```text
Un element peut etre long, frontal, rare, decoratif ou dominant si c'est decide.
Le risque stylistique apparait surtout quand il devient automatique, subi ou non situe.
```

## 1. Besoins centraux

### BS-01 - Techno comme socle recuperable

Statut : central.

Besoin :

Vesperare doit garder la techno comme socle stylistique principal, sans exiger sa presence continue en surface.

Contextes d'inspiration :

- techno fondatrice ;
- Jeff Mills : hypnose, aspiration, force de cadre ;
- Robert Hood : temps long, repetition, detail rythmique ;
- club, nuit, pression corporelle, intensite.

Signes auditifs :

- repetition hypnotique ;
- pression corporelle ;
- retour de cadre ;
- tension longue ;
- simplicite apparente mais active ;
- energie nocturne ;
- retour du corps apres suspension.

Garde-fous :

- distinguer fonction techno et timbre techno ;
- refuser les sons techno standards comme identite dominante ;
- garder pression, memoire ou direction meme quand la techno est suspendue.

### BS-02 - Techno reincarnee

Statut : central.

Besoin :

Les fonctions techno doivent pouvoir etre portees par des supports acoustiques, vocaux, instrumentaux, naturels ou pseudo-acoustiques plausibles.

Fonctions concernees :

- impact-fondation ;
- micro-pulsation ;
- pression grave ;
- masse ;
- tension ;
- retour de cadre ;
- build ;
- resolution ;
- densite d'attaques.

Supports possibles :

- didgeridoo ;
- gong ;
- tambour / peau ;
- voix ;
- bois ;
- metaux ;
- souffles ;
- frottements ;
- resonances ;
- cavites ;
- partiels.

Garde-fous :

- nommer la fonction avant la source ;
- eviter le decor naturaliste non decide ;
- verifier que le support conserve attaque, lisibilite, propulsion ou pression.

### BS-03 - Pression rythmique prioritaire

Statut : central.

Besoin :

La pression rythmique est prioritaire. Elle peut etre explicite, implicite, suspendue ou portee par l'espace.

Formes :

- pulse explicite ;
- pulse implicite ;
- interlocking ;
- syncope de masse ;
- accent fantome ;
- respiration de matiere ;
- densite d'attaques ;
- dissonance metrique ;
- rythme spatial ;
- retrait rythmique ;
- retour de cadre.

Signes de reussite :

- le corps reste oriente ;
- la tension est lisible ;
- la complexite a une direction ;
- le retrait cree une attente ;
- le retour a une valeur musicale.

Garde-fous :

- une couche directrice doit pouvoir emerger ;
- la variation doit avoir une cible ;
- la complexite ne doit pas devenir demonstration.

### BS-04 - Techno suspendue longue

Statut : central.

Besoin :

Une section peut suspendre la techno tres longtemps si elle garde pression, memoire, direction ou tension formelle.

Types :

- pulse fantome ;
- suspension spatiale ;
- suspension spectrale ;
- suspension instrumentale ;
- pre-retour ;
- faux retour ;
- temps de seuil.

Retours possibles :

- retour du grave ;
- cristallisation d'un accent ;
- pulsation qui emerge d'une texture ;
- contraction de l'espace ;
- densification d'attaques ;
- resolution microtonale ;
- impact rare.

Risques :

- espace non decide ;
- decor impose par l'outil ;
- perte d'attention ;
- retour arbitraire ;
- absence de forme non choisie.

Precision :

La techno suspendue peut durer tres longtemps.

Elle reste compatible avec Vesperare si elle conserve au moins une de ces fonctions :

- pression latente ;
- memoire ;
- direction ;
- tension formelle ;
- retrait decide du corps ;
- espace stable choisi ;
- attente de retour ;
- seuil ou hors-monde volontaire.

Le retour corporel peut etre prepare, retarde, fragmente ou refuse si cette decision est perceptible.

### BS-05 - Plausibilite acoustique et ambiguite

Statut : central.

Besoin :

Les sons electroniques reconnaissables doivent etre evites. Les sons peuvent etre ambigus ou impossibles, mais doivent rester plausibles comme extensions acoustiques.

Signes auditifs :

- cause possible : frappe, souffle, friction, pression, cavite, membrane, metal, voix, air ;
- morphologie credible : attaque, corps, queue, silence ;
- evolution organique : partiels, formants, battements, rugosite, instabilite ;
- espace plausible : distance, profondeur, surface, reflexion ;
- espace impossible decide : seuil, hors-monde, grande echelle, retrait ;
- energie corporelle : geste, effort, resistance, relachement.

Formule :

Un son peut etre impossible, mais il doit laisser imaginer une cause physique.

Garde-fous :

- eviter preset, synth lead, supersaw, pluck, wobble, riser EDM, auto-tune identifiable, glitch decoratif ;
- accepter l'ambiguite, refuser l'identification electronique evidente ;
- naturaliser pour clarifier matiere, cause, geste, espace ou fonction, pas pour ajouter du decor.

### BS-06 - Didgeridoo libre et accommodable

Statut : central pour le setup minimum.

Besoin :

Le didgeridoo doit pouvoir rester libre. Il ne doit pas etre force a devenir basse, drone, input, controleur ou role fixe.

Libertes :

- ne pas piloter ;
- ne pas jouer la basse ;
- ne pas etre traite ;
- ne pas rester en drone ;
- pouvoir devenir central ;
- pouvoir rester presence variable.

Roles possibles :

- libre ;
- protege ;
- systeme adapte au didgeridoo ;
- didgeridoo adapte au systeme ;
- grave partage ;
- basse adaptee ;
- guide harmonique ;
- harmoniques adaptees si conflit musical ;
- source traitee ;
- source fonctionnalisee ;
- perturbateur ;
- hybride.

Garde-fous :

- le contexte peut s'adapter pour qu'il reste perceptible ;
- le grave doit pouvoir se retirer, alterner ou se deplacer ;
- l'autonomie expressive doit rester prioritaire.

Modes stylistiques simples :

```text
DID-S1 - Libre protege
Le didgeridoo reste libre ; seules les protections evidentes de mix et securite interviennent.

DID-S2 - Systeme adapte au didgeridoo
Le reste de la scene suit ou respecte bourdon, partiels, energie, attaques ou souffle.

DID-S3 - Didgeridoo adapte au systeme
Le didgeridoo peut etre traite pour s'inscrire dans le grave, les harmoniques ou la scene.
```

Ces modes sont des options stylistiques, pas des obligations de jeu.

## 2. Besoins transversaux

### BS-07 - Polytexture rythmico-spectrale

Statut : transversal.

Contexte :

Keita Ogawa comme point de depart phenomenologique pour tissage, interlocking, plans, attaques et contrepoints, sans imitation.

Besoin :

La polytexture doit etre disponible comme facette operatoire, non permanente.

Elle peut etre :

- rythmique ;
- percussive ;
- spectrale ;
- harmonique ;
- vocale ;
- microtonale ;
- spatiale.

Garde-fous :

- chaque couche a un role ;
- une hierarchie perceptive reste audible ;
- pas de folklore, pas de demonstration, pas de surcharge medium.

### BS-08 - Espace et halo comme fonctions

Statut : transversal.

Besoin :

L'espace et le halo doivent etre des fonctions musicales actives, pas des effets automatiques.

Ils peuvent etre ponctuels, longs, tres longs, proches, immenses, stables, coupes, forces ou verrouilles si leur fonction est decidee.

Espace :

- profondeur ;
- distance ;
- scene ;
- separation ;
- suspension ;
- retour ;
- pression par vide ;
- confort dans l'intensite ;
- proximite ;
- immersion ;
- seuil ;
- retrait decide ;
- decor decide ;
- hors-monde.

Halo :

- prolonger ;
- lier ;
- memoriser ;
- porter des partiels ;
- ouvrir ;
- soutenir une suspension ;
- transformer une attaque en champ ;
- maintenir une fonction ;
- signaler une presence ;
- faire appartenir des sources a un monde ;
- porter une pression douce ou spectrale.

Garde-fous :

- halo long autorise seulement par fonction forte, scene ou choix live ;
- pas de grande reverb automatique non decidee ;
- grave spatial seulement selon mode clair : Off, Trace, Corps, Pression, Bloom ou Libre ;
- les attaques principales doivent rester lisibles.

Modes stylistiques :

```text
HAL-S1 - Auto discret
Le halo sert la coherence, l'espace commun ou la douceur.

HAL-S2 - Lock
Le halo reste perceptible car il porte memoire, suspension, pression ou monde.

HAL-S3 - Force
Le halo devient decision forte, meme s'il prend de la place.

HAL-S4 - Cut
Le halo est retire volontairement pour retour du corps, nudite, impact ou clarification.

HAL-S5 - Momentary
Le halo transforme un geste, une attaque, une queue ou un seuil.
```

Regle :

```text
Le probleme n'est pas la duree du halo.
Le probleme est un halo non decide qui floute la fonction musicale.
```

Ambient :

Vesperare n'est pas interdit d'espace long, de vide, de decor ou d'immersion.

Le risque stylistique est l'ambient subi :

- espace qui remplace la forme sans decision ;
- grande queue automatique ;
- flou qui annule pression, source ou retour ;
- profondeur confortable sans enjeu ;
- halo qui devient fond par defaut.

### BS-09 - Melodie variable et naturalisee

Statut : transversal.

Besoin :

Vesperare n'est pas anti-melodique. La melodie doit pouvoir apparaitre sous des formes liees a la matiere, au souffle, au grave, a la voix, aux partiels ou aux accents.

Regimes :

- absence de melodie ;
- centre ressenti ;
- accent-melody ;
- basse-motif ;
- melodie de timbre ;
- melodie de partiels ;
- ligne acid naturalisee ;
- fragment vocal ;
- hook minimal ;
- melodie microtonale.

Garde-fous :

- pas de lead banal ;
- pas de hook generique ;
- pas de ligne acid autonome ;
- toute melodie doit avoir une source ou une fonction.

### BS-10 - Microtonalite, partiels, pitch large

Statut : transversal.

Besoin :

La microtonalite doit organiser couleur, tension, battements, partiels, formants, hauteurs ressenties, centres spectraux et resolutions.

Regimes :

- par defaut : couleur subtile, realisme acoustique, battements ;
- ponctuel : microtonalite audible et structurante.

Garde-fous :

- pas de demonstration de tuning ;
- pas de gamme exotique plaquee ;
- toujours relier microtonalite a tension, resolution, naturalisation ou expressivite.

### BS-11 - Scene auditive et hierarchie perceptive

Statut : transversal.

Besoin :

Vesperare doit etre pense comme scene auditive : plans, fusion, separation, masquage, profondeur, proximite, stabilite et emergence d'objets.

Axes :

- objet sonore ;
- texture sonore ;
- entre-deux ;
- avant-plan ;
- fond ;
- espace ;
- grave ;
- presence instrumentale.

Garde-fous :

- ne pas tout isoler en objets ;
- ne pas tout dissoudre en texture ;
- preserver une hierarchie perceptive ;
- surveiller les monopoles : sub, articulation, brillant continu, halo long, transgression.

Precision :

Ces monopoles sont des garde-fous perceptifs, pas des interdictions absolues.

Ils peuvent etre transgresses par :

- choix live explicite ;
- scene preparee ;
- fonction forte ;
- transgression volontaire ;
- retrait decide ;
- hors-monde decide ;
- graves libres ;
- halo Lock ou Force.

Limites :

- securite critique ;
- sub illisible non voulu ;
- attaque principale masquee sans fonction forte ;
- didgeridoo, voix ou gong essentiels effaces sans decision.

### BS-12 - Intensite sans agression

Statut : transversal.

Besoin :

Vesperare doit pouvoir atteindre une intensite enorme, parfois transgressive, sans s'appuyer sur l'agression sonore gratuite.

Sources :

- pression rythmique ;
- pression grave ;
- pression spectrale ;
- espace immense ;
- rarete ;
- retour de cadre ;
- rugissement ;
- cri ;
- saturation organique ;
- microtonalite exposee ;
- contraste nudite/masse.

Garde-fous :

- identifier la source de l'intensite ;
- prevoir une sortie, un maintien decide ou une consequence ;
- distinguer intensite, transgression, pic et agression ;
- garder le confort comme contrainte musicale.

### BS-13 - Beaute fonctionnelle et beaute autonome

Statut : transversal.

Besoin :

Vesperare doit permettre la beaute fonctionnelle et certains moments de beaute presque autonome.

Beaute fonctionnelle :

- un son est beau parce qu'il est exactement a sa place.

Beaute autonome :

- moment de matiere ;
- voix exposee mais integree acoustiquement ou virtualo-acoustiquement ;
- gong expose ;
- champ de partiels ;
- suspension ;
- halo ;
- microtonalite perceptible.

Garde-fou :

Un moment autonome doit garder pression, attente, transition, exposition, transgression, memoire ou direction.

## 4. Garde-fous stylistiques globaux

### BS-20 - Anti-collage

Statut : garde-fou.

Regle :

Les inspirations peuvent se superposer ou etre absentes, mais elles ne doivent pas devenir des calques stylistiques.

Question correcte :

- quelle fonction est active ?
- par quel support ?
- avec quelle pression ?
- avec quelle plausibilite acoustique ?
- avec quel risque ?
- avec quel garde-fou ?

### BS-21 - Anti-glissement

Statut : garde-fou.

Glissements a eviter :

- intention -> parametre technique ;
- techno organique -> collection d'effets ;
- didgeridoo -> source sub ;
- hypnose -> randomisation ;
- complexite -> accumulation ;
- naturalisation -> decor non decide ;
- melodie -> lead banal ;
- espace -> ambient subi ;
- halo Lock -> impossibilite de proteger ;
- grave libre -> boue grave ;
- monde spatial -> preset de reverb ;
- source plausible -> illustration naturaliste.

Regle :

```text
Le risque stylistique n'est pas l'existence d'un element.
Le risque est son autonomie non decidee, son automatisme ou son absence de fonction.
```
