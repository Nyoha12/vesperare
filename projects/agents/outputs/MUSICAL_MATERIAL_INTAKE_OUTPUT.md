## Sources concrètes

Faits présents dans les fichiers :

- Sources live fixes :
  - didgeridoo live ;
  - guimbardes live.
- Sources live optionnelles :
  - gong ;
  - tambour, notamment tambour sur cadre / peau ;
  - voix, seulement si sa fonction est validée.
- Sources ou supports possibles décrits :
  - sub discret, sub retenu, sub latent, sub stable ;
  - synthèse grave minimale ou synthèse à comportement acoustique ;
  - impact acoustique, objet frappé, percussion profonde, peau grave ;
  - micro-percussions, objets courts, bois frappés, petits métaux, cymbalettes, cloches, clochettes, shakers naturels ;
  - bols chantants, résonances naturelles, halos, résonateurs spectraux ;
  - voix parlée, souffle, mot isolé, voix coupée, voix filtrée, cri, fragment, signal ;
  - field recordings, mais seulement conditionnels et avec fonction claire ;
  - loop temporaire comme trace, repeat rythmique contrôlé, freeze bref avec release.
- Sources à éviter ou à tenir à distance au départ :
  - field recording décoratif ;
  - halo large permanent ;
  - freeze long ;
  - loop de remplacement ;
  - texture d'ambiance ;
  - polytexture automatisée trop pleine ;
  - gong spectaculaire immédiat ;
  - voix criée non préparée ;
  - repeat démonstratif.

Inférences prudentes :

- Les sources les plus directement disponibles pour une pensée musicale sont les instruments live centraux : didgeridoo et guimbardes.
- Le tambour, le gong, la voix, le field recording, le sub et les objets courts sont des candidats fonctionnels, mais leur statut dépend de scènes et de rôles précis.
- Le sub est décrit comme appui, contrepoids, tension, stabilisation ou résolution possible ; il n'est pas décrit comme source autonome validée.

Incertitudes :

- Le corpus lu ne confirme pas quelles prises audio existent réellement.
- Le corpus lu ne donne pas les modèles précis d'instruments, accordages, tessitures, micros, captations, chaînes de traitement ou contraintes de diffusion.
- Le gong, le tambour et la voix restent conditionnels : leur présence réelle, leur rôle et leur degré de centralité ne sont pas décidés par les fichiers lus.

## Gestes décrits

Faits présents dans les fichiers :

- Didgeridoo :
  - tenir le sol ;
  - varier la pression ;
  - ouvrir ou fermer les formants ;
  - créer une poussée sans forcer le drop ;
  - devenir pression incomplète ;
  - devenir trace, souffle, centre grave ou partiel ;
  - préparer le retour au corps sans l'imposer ;
  - quitter le rôle de sol pour devenir tension ou trace.
- Guimbardes :
  - créer micropulse ;
  - activer la bouche comme filtre vivant ;
  - introduire des asymétries ;
  - percer la densité sans devenir ornement ;
  - fragmenter le rythme ;
  - créer impatience, ressort, seuil ;
  - espacer, scintiller, tracer des micro-accents ;
  - rester audibles comme gestes, pas comme décor.
- Sub :
  - soutenir discrètement ;
  - se retenir ;
  - pousser sans exploser ;
  - se retirer ;
  - revenir comme rappel ;
  - stabiliser ou résoudre ;
  - rester centre physique sans couvrir le didgeridoo.
- Tambour :
  - renforcer l'ancrage ;
  - poser une peau dans le corps ;
  - préparer ou servir un retour au corps ;
  - donner une résolution sèche plutôt qu'un drop massif.
- Gong :
  - ponctuer ;
  - ouvrir un seuil ;
  - laisser une masse courte ;
  - être armé, retardé, presque entendu avant d'être frappé ;
  - rugir ou résoudre ;
  - ouvrir un champ de partiels ;
  - laisser une queue contrôlée ;
  - fermer ou faire basculer une suspension.
- Voix :
  - souffle ;
  - fragment ;
  - signal ;
  - cri possible ;
  - présence humaine ;
  - jamais effet gratuit.
- Gestes temporels :
  - repeat court si fonction rythmique, corporelle ou dramatique claire ;
  - loop comme trace, ombre, double ou transition ;
  - freeze très bref avec release obligatoire ;
  - silence court ou trou de densité comme seuil.
- Opérations dramaturgiques :
  - retirer ;
  - retenir ;
  - densifier ;
  - accentuer ;
  - fragmenter ;
  - tendre ;
  - armer ;
  - retarder ;
  - résoudre ;
  - bifurquer ;
  - suspendre ;
  - exposer ;
  - réduire ;
  - réincorporer ;
  - transférer.

Inférences prudentes :

- Les gestes sont décrits comme fonctions de jeu et d'écoute, pas comme gestes techniques détaillés.
- Le didgeridoo et les guimbardes sont pensés comme agents de scène, pas comme simples couches sonores.
- Les gestes de retrait, trace, seuil et retour sont aussi importants que les gestes d'ajout ou d'intensification.

Incertitudes :

- Les fichiers ne disent pas quels gestes sont déjà maîtrisés, enregistrés ou disponibles dans le jeu actuel de Yohan.
- Les descriptions de formants, partiels, pression et micropulse restent qualitatives.

## Scènes / situations décrites

Faits présents dans les fichiers :

- Scènes-champs ouvertes :
  - établir ou rétablir le corps ;
  - construire pression et énergie sans perdre lisibilité ;
  - densité polytexturale lisible et corporelle ;
  - attente / non-résolution / tension longue ;
  - pré-drop armé sans obligation de résolution ;
  - suspension / expansion sans dissolution du corps ;
  - retrait / respiration / révélation ;
  - retour au corps ;
  - matière exposée.
- Trois scènes-pivots concrètes :
  - SCENE_PIVOT_CORPS_VIVANT_POLYTEXTURE ;
  - SCENE_PIVOT_TENSION_ARMEE_PRE_DROP ;
  - SCENE_PIVOT_SUSPENSION_MATIERE_RETOUR.
- Dramaturgie corps vivant / polytexture :
  - commencer par un contact, pas une boucle pleine ;
  - faire apparaître le corps avant le beat complet ;
  - développer par articulation, hiérarchie d'accents, respiration du sub, mouvement des guimbardes ;
  - ouvrir vers pression, pré-drop, suspension, retour au corps ou trace temporelle courte.
- Dramaturgie tension armée / pré-drop :
  - commencer par un retrait ;
  - armer une résolution sans l'imposer ;
  - laisser plusieurs issues possibles : résolution, fausse résolution, annulation, suspension, retour au corps ;
  - garder la tension comme qualité changeante, pas comme attente statique.
- Dramaturgie suspension / matière / retour :
  - retirer sans arrêter ;
  - exposer une matière comme fonction, pas comme décor ;
  - tenir la suspension avec direction ;
  - garder une sortie audible ;
  - réincorporer ou disparaître avec trace.
- Transitions décrites :
  - corps vivant vers tension armée ;
  - tension armée vers suspension ;
  - suspension vers corps vivant ;
  - suspension vers tension armée.

Inférences prudentes :

- Les scènes les plus prêtes musicalement sont les trois scènes-pivots, car elles incluent sources, gestes, phases, risques, points de contrôle et critères d'écoute.
- Les autres champs de scène sont utiles comme paysage fonctionnel, mais moins incarnés dans des gestes de jeu.

Incertitudes :

- Les scènes ne sont pas encore appuyées par des prises réelles.
- Les durées indiquées sont indicatives et ne prouvent pas que la dramaturgie tiendra en performance.
- Les relations entre scènes restent des possibilités décrites, pas des transitions validées musicalement.

## Problèmes sonores concrets

Faits présents dans les fichiers :

- Masquage :
  - sub qui couvre le didgeridoo ;
  - guimbardes masquées ;
  - voix ou halos masqués par des aigus ou de l'espace ;
  - objets centraux masqués par un halo permanent ;
  - gongs ou bas-médiums créant de la boue spectrale.
- Grave et pression :
  - grave flou ;
  - sub instable ;
  - pression confondue avec volume ;
  - bas-médium boueux ;
  - pression qui écrase les détails ;
  - perte de mobilité scénique.
- Rythme et corps :
  - attaque molle ;
  - perte de danse ;
  - répétition morte ;
  - repeat spectaculaire gratuit ;
  - loop qui remplace le live ;
  - densité qui perd la hiérarchie ;
  - polytexture confuse.
- Espace et halo :
  - ambientisation ;
  - halo permanent ;
  - freeze-drone ;
  - reverb / espace décoratif ;
  - perte d'impact ;
  - flou rythmique ;
  - fatigue par saturation d'espace.
- Timbre et matière :
  - son trop sec ou mort ;
  - queue non contrôlée ;
  - résonance qui fait perdre la fonction ;
  - matière fascinante mais sans conséquence formelle ;
  - source trop identifiable sans rôle.
- Forme :
  - drop automatique ;
  - pré-drop trop reconnaissable comme recette ;
  - résolution qui efface la tension construite ;
  - retour au corps arbitraire ou brutal ;
  - suspension qui devient pause ;
  - beauté qui coupe la forme.
- Live / agency :
  - didgeridoo réduit à une basse utilitaire ;
  - guimbardes devenues décor ethnique ou métallique ;
  - système imaginaire qui semble jouer à la place du corps ;
  - live devenu couleur plutôt que source d'énergie.

Inférences prudentes :

- Les problèmes les plus concrets se concentrent autour de trois zones : lisibilité du corps, protection des sources live, contrôle du grave / halo / temporalité.
- Le corpus signale déjà que les risques techniques et les risques esthétiques sont liés : un problème de queue, de grave ou de densité peut devenir une perte de style.

Incertitudes :

- Aucun exemple sonore réel ne permet de savoir quelle zone est actuellement la plus critique dans le jeu ou le mix.
- Les seuils perceptifs de masquage, fatigue, boue et stabilité ne sont pas quantifiés.

## Critères perceptifs

Faits présents dans les fichiers :

- Corps :
  - le corps doit être évident sans être plat ;
  - il peut être latent, mais pas perdu ;
  - il doit être présent avant le beat complet dans la scène corps vivant ;
  - le retour au corps doit rester possible.
- Sources live :
  - les sources live doivent rester centrales ;
  - didgeridoo et guimbardes doivent rester actifs comme agents ;
  - le live ne doit pas devenir décor.
- Pression :
  - la pression doit être physique et lisible ;
  - le sub doit soutenir sans dominer ;
  - l'intensité ne doit pas dépendre principalement du volume ou de l'agression des aigus.
- Polytexture :
  - la densité doit rester articulée ;
  - les accents doivent être hiérarchisés ;
  - la complexité doit augmenter l'énergie sans devenir démonstrative.
- Pré-drop / tension :
  - la résolution doit rester désirable mais incertaine ;
  - plusieurs sorties doivent rester crédibles ;
  - le seuil doit être musical, pas seulement technique ;
  - l'attente doit changer de qualité.
- Suspension :
  - l'écoute doit devenir plus proche ;
  - la matière exposée doit avoir une fonction ;
  - la sortie doit rester audible ;
  - le silence relatif ou l'espace ne doivent pas devenir arrêt ;
  - le halo doit servir la forme.
- Contour :
  - attaque lisible ;
  - queue maîtrisée ;
  - enveloppe claire ;
  - richesse lisible ;
  - objet gardant sa personnalité sans perdre son rôle.
- Temporalité :
  - repeat avec fonction ;
  - loop comme trace, non remplacement ;
  - freeze avec release ;
  - sortie possible.

Inférences prudentes :

- Les critères perceptifs sont assez développés pour guider une validation humaine.
- Ils restent majoritairement qualitatifs et demandent l'écoute de Yohan pour devenir discriminants.

Incertitudes :

- Les fichiers ne donnent pas de seuils de durée, niveaux, fréquences, densité maximale ou amplitude de transformation.
- Les critères de réussite sont décrits, mais pas encore reliés à des exemples sonores.

## Risques / pertes décrits

Faits présents dans les fichiers :

- Perte du corps :
  - ambientisation ;
  - suspension sans sortie ;
  - halo permanent ;
  - field recording décoratif ;
  - beat ou pulsation absents sans trace corporelle.
- Perte de live :
  - loop qui remplace le geste ;
  - didgeridoo utilitaire ;
  - guimbardes décoratives ;
  - automation ou système qui prend la place de l'agency musicale.
- Perte de fonction techno :
  - texture complexe sans hiérarchie ;
  - beauté autonome sans conséquence formelle ;
  - drop standard ;
  - pré-drop reconnaissable ;
  - répétition morte ;
  - ornement sans propulsion.
- Perte de lisibilité :
  - polytexture confuse ;
  - trop de couches équivalentes ;
  - médiums chargés ;
  - queues non contrôlées ;
  - grave flou.
- Perte de singularité :
  - kick électronique trop standard ;
  - citation acid trop reconnaissable ;
  - supersaw / riser / effet de style plaqué ;
  - voix gimmick ;
  - field recording naturaliste sans rôle.
- Perte de forme :
  - tension qui ne change plus ;
  - résolution qui efface tout ;
  - retour arbitraire ;
  - freeze qui devient drone ;
  - silence vide au lieu de silence chargé.

Inférences prudentes :

- Le risque central n'est pas seulement de "mal sonner", mais de perdre l'intégrité du jeu : corps, live, tension, forme et agency.
- Les pertes décrites sont des alertes perceptives plus que des interdits absolus.

Incertitudes :

- Certaines limites peuvent devenir expressives si Yohan les assume musicalement ; le corpus demande de les traiter comme zones de bord, pas comme murs fixes.

## Directions positives décrites

Faits présents dans les fichiers :

- Techno naturalisée :
  - conserver corps, impact, pression, répétition, hypnose, tension et forme longue ;
  - changer les supports sonores sans perdre les fonctions.
- Centralité live :
  - didgeridoo et guimbardes comme sources fixes centrales ;
  - live comme source d'énergie, pas couleur ajoutée.
- Corps vivant :
  - pression lisible ;
  - micropulsation active ;
  - sub stable mais non dominateur ;
  - densité mobile ;
  - hypnose par articulation plutôt que remplissage.
- Tension armée :
  - résolution possible mais non obligatoire ;
  - attente corporelle ;
  - seuils ouverts ;
  - annulation et fausse résolution comme sorties musicales possibles.
- Suspension matière retour :
  - matière exposée avec fonction ;
  - corps latent ;
  - écoute rapprochée ;
  - trace, retrait, retour ou bifurcation.
- Sound design :
  - intensité phénoménale sans agression ;
  - contour sculpté ;
  - queue contrôlée ;
  - richesse lisible ;
  - beauté autonome placée et reliée à la forme.
- Influences intégrées :
  - dubstep comme tension / attente / résolution, pas style importé ;
  - jazz comme champs harmoniques, couleurs et finesse rythmique, pas jazz reconnaissable par défaut ;
  - trance comme suspension / ouverture / peak non brutal ;
  - acid comme torsion, ligne vivante et résonance contrôlée, pas imitation TB-303.

Inférences prudentes :

- Les directions positives les plus incarnées sont celles où une source live porte une fonction perceptible : didgeridoo-pression, guimbardes-micropulse, gong-seuil, tambour-retour, sub-contrepoids.
- La force du corpus est de formuler des situations avec sorties, pas seulement des atmosphères.

Incertitudes :

- Les directions positives restent à confirmer par l'écoute et le jeu.
- Certaines directions, comme voix centrale, field recording, harmonic drone, cordes ou métaux accordés, restent ouvertes mais non établies.
