## Apparent project type

Vesperare is currently a research-creation and documentation repository for the future design of a live composition/performance system in Ableton Live / Max for Live.

It is not currently a Max for Live implementation repository.

## Apparent goal

The apparent goal is to progressively translate an artistic and performative intention into:
- concepts;
- source needs;
- system modules;
- control logic;
- future verifiers;
- eventually targeted Max for Live prototypes.

The artistic direction appears to center on a personal genre called "techno organique", but this must later be treated as a claim to audit, not as proof.

## Current phase

The repository appears to be in:

conception / consolidation / harmonization before prototyping.

This phase must not automatically lead to:
- patch;
- layout;
- mapping;
- prototype;
- engine profile;
- real audio asset;
- new source_need;
- new scene.
