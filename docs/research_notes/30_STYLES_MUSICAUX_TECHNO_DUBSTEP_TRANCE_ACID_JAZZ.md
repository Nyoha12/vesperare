# 30_STYLES_MUSICAUX_TECHNO_DUBSTEP_TRANCE_ACID_JAZZ

Version : v0.1  
Statut : notes de recherche externes sur styles musicaux. Réserve documentaire extractible, pas source de vérité esthétique.

## Objet

Stocker les premiers résultats de recherche musicale utiles pour réancrer les notions de Vesperare dans des contextes musicaux précis.

Principe : ne pas importer les genres. Extraire seulement des fonctions, phénomènes, risques et questions utiles pour Vesperare.

## 1. Techno / minimal techno

### Sources initiales

- Robert Hood / minimal techno / Paradygm Shift : `https://pitchfork.com/reviews/albums/23380-paradygm-shift`
- Minimal Nation : `https://en.wikipedia.org/wiki/Minimal_Nation`
- Techno minimale : `https://fr.wikipedia.org/wiki/Techno_minimale`

### Observations brutes

Les sources sur Robert Hood et la minimal techno mettent en avant :

- réduction ;
- répétition ;
- efficacité ;
- boucles arpégées ;
- micro-variations ;
- mutations rythmiques ;
- rythmes imbriqués ;
- groove maintenu par de petits déplacements ;
- tension mécanique mais vivante.

### Traduction Vesperare possible

À relier à :

- répétition non redondante ;
- hypnose active ;
- contour sculpté ;
- pression sans surcharge ;
- corps par réduction ;
- intensité sans agression ;
- micro-variation fonctionnelle.

### Risques de caricature

- techno = kick + sub ;
- minimal = vide ;
- répétition = boucle figée ;
- variation = randomisation visible ;
- pression = volume ;
- efficacité = brutalité.

Recherches à poursuivre :

- Jeff Mills / The Bells / tension répétitive ;

- minimal techno comme réduction de fonctions ;
- différence répétition hypnotique / boucle décorative.

## 5. Jazz

### Sources initiales

- Comping jazz : `https://en.wikipedia.org/wiki/Comping_(jazz)`
- Jazz chord : `https://en.wikipedia.org/wiki/Jazz_chord`
- Jazz harmony : `https://en.wikipedia.org/wiki/Jazz_harmony`
- Altered chord : `https://en.wikipedia.org/wiki/Altered_chord`
- Suspended chord : `https://en.wikipedia.org/wiki/Suspended_chord`
- Microtiming and swing : `https://arxiv.org/abs/1904.03442`
- Correlated microtiming deviations : `https://arxiv.org/abs/1710.05608`

### Observations brutes

Les sources indiquent :

- comping = accompagnement, complément, soutien de solo ou ligne mélodique ;
- comping moderne = plus d'espace, moments rythmiquement intéressants ;
- accords avec 3e, 7e, 9e, 13e, altérations ;
- voicings souvent sans fondamentale ou quinte ;
- extensions et couleurs ajoutées selon contexte ;
- accords suspendus = ambiguïté, tension non toujours résolue ;
- swing ne doit pas être réduit mécaniquement à microtiming ;
- microtiming existe mais ne suffit pas à définir le swing.

### Traduction Vesperare possible

À relier à :

- champs de tension ;
- couleurs d'accords ;
- voicings distribués ;
- harmonie comme matière ;
- harmonie non décorative ;
- tension suspendue ;
- placement rythmique ;
- accentuation fine ;
- rapport harmonie / rythme ;
- comping comme soutien, non remplissage.

### Risques de caricature

- jazz = accords riches ;
- jazz = humanize ;
- swing = microtiming aléatoire ;
- harmonie = décoration ;
- voicing = couleur plaquée ;
- comping = remplissage ;
- tension = grille harmonique importée.

### À intégrer dans Vesperare

Documents internes probables :

- `docs/06_CARTOGRAPHIE_INFLUENCES_STYLE_VESPERARE.md`
- `docs/07_PITCH_HARMONIQUES_MICROTONALITE.md`
- `docs/01_CAHIER_DES_PHENOMENES_SONORES.md`

Recherches à poursuivre :

- voicing distribué hors clavier ;
- comping comme écoute et soutien ;
- tension non résolue ;
- swing comme relation globale plutôt que déviation aléatoire ;
- rapport harmonie/rythme dans texture électronique.
