# 09_TARGETED_ADDITIONAL_INSTRUMENTS_FROM_REMAINING_NEEDS

Version : v0.1  
Statut : analyse ciblée. Sans ajout validé, sans sample confirmé, sans prototypage.

## Objet

Ce document rouvre les instruments / sources candidates supplémentaires, mais uniquement à partir des manques identifiés dans :

```text
08_REMAINING_NEEDS_AFTER_LIVE_BASE_AND_COMPENSATIONS
```

Il ne sert pas à dresser une liste d’instruments séduisants.

Il sert à répondre à la question :

```text
quelle source pourrait couvrir un besoin encore mal couvert,
sans remplacer inutilement didgeridoo, guimbardes, sub, gong, tambour,
et sans créer une dérive esthétique ?
```

---

# 1. Règle de sélection

Une source supplémentaire ne peut devenir pertinente que si elle répond à au moins une de ces conditions :

```text
elle couvre un besoin encore ouvert ;
elle couvre mieux qu’une source déjà présente ;
elle apporte une fonction impossible à obtenir par la base live + compensations ;
elle réduit un risque au lieu d’en ajouter ;
elle reste compatible avec le corps techno, la pression, l’hypnose et la naturalisation ;
elle peut être intégrée comme source, sample minimal, engine, traitement ou référence claire.
```

Elle doit être rejetée ou reportée si :

```text
elle double une fonction déjà forte ;
elle ajoute seulement une couleur ;
elle crée une dérive décorative ;
elle rend le système plus complexe sans fonction ;
elle folklorise ou théâtralise le matériau ;
elle éloigne le projet du corps techno.
```

---

# 2. Besoins ouverts prioritaires

D’après la matrice précédente, les besoins les plus ouverts sont :

```text
champs harmoniques soutenus ;
battements / microtonalité élargie ;
expansion / suspension non ambient ;
ligne de torsion médium ou aiguë ;
voix humaine réelle si scène l’exige ;
lieu réel si scène l’exige ;
éléments naturels procéduraux ou réels si fonctionnels ;
résonances pincées / frappées organiques ;
métaux accordés / cloches / vibraphone.
```

---

# 3. Sources candidates ciblées

## 3.1 Shruti box / harmonium / bourdon portatif

```text
besoin visé: champ harmonique soutenu, battements, suspension, souffle harmonique.
fonction possible: masse harmonique respirante, tension longue, drone non électronique, battements microtonaux.
mode possible: live, sample minimal, référence, traitement, resampling.
priorité: haute.
compatibilité: forte si traité comme champ de battements et non comme décor méditatif.
risques: ambientisation, spiritualité plaquée, drone statique, perte de tension techno.
conditions d’usage: uniquement si le bourdon produit tension, battement, suspension active ou champ harmonique utile.
raison de rejet si base suffit: didgeridoo + sub + résonateurs couvrent déjà le grave/souffle ; inutile si le besoin harmonique large n’est pas confirmé.
```

## 3.2 Orgue / orgue d’église / orgue de cathédrale

```text
besoin visé: architecture harmonique, masse, espace, battements, expansion monumentale.
fonction possible: champ harmonique massif, espace architectural, tension longue, beauté autonome contrôlée.
mode possible: sample minimal, field recording instrumental, convolution/empreinte, référence ; live peu probable.
priorité: haute mais dangereuse.
compatibilité: forte si utilisé comme matière/architecture, faible si utilisé comme grandeur sacrée.
risques: décor sacré, majesté trop évidente, ambientisation, cliché cathédrale, effacement du corps techno.
conditions d’usage: très peu de prises, rôle structurel clair, pas de nappes décoratives.
raison de rejet si base suffit: si shruti/résonateurs/gongs couvrent déjà suspension et harmoniques, l’orgue peut être trop lourd.
```

## 3.3 Cordes frottées

```text
besoin visé: torsion médium/aiguë, tension, friction, lignes vivantes, battements.
fonction possible: tension mobile, rugosité contrôlée, ligne organique, microtonalité expressive, cri sans voix.
mode possible: sample minimal, live invité éventuel, resampling, traitement, référence.
priorité: haute pour la ligne de torsion.
compatibilité: forte si utilisé comme matière tendue et non comme geste romantique.
risques: lyrisme, pathos, musique de film, surcharge émotionnelle, perte de sécheresse sculptée.
conditions d’usage: attaques, frottements, tenues instables, intervalles/tensions, pas de phrases expressives classiques.
raison de rejet si base suffit: si didgeridoo/guimbardes/résonateurs suffisent à la torsion, les cordes peuvent ajouter du pathos inutile.
```

## 3.4 Anches / hautbois / basson / vents à anche

```text
besoin visé: voix-sans-voix médium/aiguë, cri, tension, formants, ligne vivante.
fonction possible: signal non vocal, cri retenu, couleur formantique, ligne de torsion, tension respirée.
mode possible: sample minimal, live invité éventuel, référence, resynthesis/traitement.
priorité: moyenne à haute.
compatibilité: forte si la source sert le cri, le souffle ou la torsion ; faible si elle devient couleur orchestrale.
risques: orchestration, drame, effet solo, doublon avec voix/guimbardes/didgeridoo.
conditions d’usage: fragments, souffles, attaques, notes tenues instables, formants exposés.
raison de rejet si base suffit: guimbardes + didgeridoo couvrent déjà bouche/formants ; anches nécessaires seulement pour registre ou cri non couvert.
```

## 3.5 Cuivres / trompettes / souffles métalliques

```text
besoin visé: signal, cri, tension, projection, halo métallique soufflé.
fonction possible: signal formel, cri non vocal, accent de tension, matière soufflée brillante, appel.
mode possible: sample minimal, live invité éventuel, traitement, référence.
priorité: moyenne.
compatibilité: forte si utilisé comme signal/pression/tension, faible si héroïsme ou fanfare.
risques: dramatique plaqué, effet trailer, jazz/blues/fanfare trop identifiable, brillance punitive.
conditions d’usage: attaques rares, souffles, sourdines, multiphoniques, fragments non mélodiques.
raison de rejet si base suffit: voix conditionnelle, gong et guimbardes peuvent déjà porter signal ; cuivre à réserver aux besoins de projection non couverts.
```

## 3.6 Flûtes / souffles non anches

```text
besoin visé: air, souffle, tension légère, halo, fragilité, microtexture respirée.
fonction possible: souffle aigu/médium, respiration, trace aérienne, contraste avec didgeridoo grave.
mode possible: sample minimal, live invité éventuel, traitement, référence.
priorité: basse à moyenne.
compatibilité: moyenne ; utile si manque d’air haut ou de souffle non grave.
risques: décor pastoral, nature plaquée, flûte trop identifiable, perte d’intensité.
conditions d’usage: souffles, air-tones, attaques, bruits, pas de mélodie décorative.
raison de rejet si base suffit: didgeridoo couvre souffle grave ; field/génération peuvent couvrir air/halo.
```

## 3.7 Vibraphone

```text
besoin visé: métal accordé, partiels contrôlés, beauté autonome, signaux harmoniques, microtensions.
fonction possible: ponctuation harmonique, résonance accordée, halo court, brillance contrôlée.
mode possible: sample minimal, resampling, traitement, référence.
priorité: moyenne à haute.
compatibilité: forte si traité comme métal harmonique précis ; faible si couleur jazz/lounge/féerique.
risques: jazz décoratif, féérie, carillon, douceur trop illustrative, perte de pression.
conditions d’usage: notes rares, résonances préparées, traitements, intervalles microtonaux ou partiels choisis.
raison de rejet si base suffit: gong/guimbardes couvrent métal macro/micro ; vibraphone seulement si besoin de métal accordé stable.
```

## 3.8 Cloches / cloches tubulaires / bols / métaux accordés

```text
besoin visé: partiels, signaux, beauté autonome, résonance longue, microtensions.
fonction possible: signal harmonique, halo métallique, marqueur formel, champ de partiels.
mode possible: sample minimal, live ponctuel, traitement, résonateur, référence.
priorité: moyenne à haute.
compatibilité: forte si chaque son a un rôle formel ; faible si carillon décoratif.
risques: new age, rituel plaqué, cloche illustrative, surcharge métallique avec gong/guimbardes.
conditions d’usage: peu d’éléments, sélection par fonction, relation au pitch_center.
raison de rejet si base suffit: gong + guimbardes suffisent si aucun besoin de métal accordé ou signal stable.
```

## 3.9 Harpes / guitares / cordes pincées

```text
besoin visé: résonances pincées, attaques organiques, beauté fragile, tension harmonique courte.
fonction possible: ponctuation, résonance courte, corps fragile, texture tendue, contraste avec métal/peau.
mode possible: sample minimal, live invité éventuel, objets tendus, traitement, référence.
priorité: moyenne.
compatibilité: moyenne à forte si utilisé comme matière pincée et non comme instrument reconnaissable.
risques: folk, guitare décorative, harpe féerique, harmonie trop tonale, éloignement techno.
conditions d’usage: attaques isolées, préparations, désaccords, résonances courtes, traitements.
raison de rejet si base suffit: inutile si les champs harmoniques ou la beauté autonome sont couverts par métaux/orgue/shruti.
```

## 3.10 Percussions supplémentaires / objets frappés

```text
besoin visé: impact corporel si tambour absent, attaques non standardisées, grains de matières.
fonction possible: peau alternative, bois, pierre, métal court, objet corporel, accent.
mode possible: sample minimal, live ponctuel, enregistrement ciblé, traitement, référence.
priorité: moyenne à haute si tambour absent souvent.
compatibilité: forte si fonctionnelle et non folklorisante.
risques: banque de percussions, tribalisation, remplissage, perte de singularité.
conditions d’usage: très peu de sources, choisies par attaque/corps/rôle, pas par couleur.
raison de rejet si base suffit: si tambour live ou compensation de peau couvre les besoins, ne pas ajouter.
```

## 3.11 Éléments naturels procéduraux / eau / pluie / vent / feu / orage

```text
besoin visé: micro-événements, flux, écologie sonore, transitions, matière environnementale.
fonction possible: texture active, système génératif, densité, bruit naturel contrôlé, événement non instrumental.
mode possible: génération/procédural en priorité, grains réels ou field recordings si exposés.
priorité: moyenne.
compatibilité: forte si fonctionnel et organisé comme écosystème ; faible si bruitage nature.
risques: décor naturaliste, démonstration sound design, ambientisation, carte postale sonore.
conditions d’usage: liens clairs aux scènes, densité contrôlée, intégration au rythme/pression/espace.
raison de rejet si base suffit: si résonateurs/field/minimal textures suffisent, ne pas multiplier les éléments naturels.
```

# 4. Priorités de recherche proposées

## Priorité haute

```text
shruti / harmonium / bourdon portatif ;
orgue comme architecture harmonique, avec prudence ;
cordes frottées pour torsion ;
métaux accordés / cloches / vibraphone pour partiels contrôlés.
```

## Priorité moyenne

```text
anches / basson / hautbois ;
cuivres / trompettes comme signaux ;
percussions supplémentaires / objets frappés ;
harpes / cordes pincées ;
élements naturels procéduraux.
```

## Priorité basse ou très conditionnelle

```text
flûtes si besoin de souffle haut spécifique ;
guitares reconnaissables ;
instruments très connotés sans transformation forte ;
sources qui doublent trop didgeridoo ou guimbardes.
```

# 5. Prochaine action recommandée

Créer des fiches `source_need` ciblées seulement pour les priorités hautes :

```text
SOURCE_NEED_HARMONIC_DRONE_SHRUTI_ORGAN
SOURCE_NEED_BOWED_STRING_TORSION_FIELD
SOURCE_NEED_TUNED_METAL_PARTIALS_BELLS_VIBRAPHONE
```

Ensuite seulement : décider si ces sources doivent devenir :

```text
live source ;
sample minimal ;
engine/résonateur ;
référence de sound design ;
source à rejeter / reporter.
```
