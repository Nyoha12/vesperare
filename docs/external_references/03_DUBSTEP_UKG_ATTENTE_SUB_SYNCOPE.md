# 03_DUBSTEP_UKG_ATTENTE_SUB_SYNCOPE

Version : v0.1  
Statut : références externes spécialisées. Non source de vérité Vesperare.

## Objet

Stocker les premiers résultats utiles pour contextualiser dubstep, UK garage, 2-step, attente, sub, espace, syncope et résolution rare.

But : éviter de réduire le dubstep au drop spectaculaire ou au cliché brostep.

---

# Synthèse rapide pour extraction

## Apports utiles

```text
dubstep = syncope, sub, espace, sparsité, sound system, tension ;
2-step = suppression / déplacement de battements, intérêt maintenu par syncope et lignes de basse ;
UK garage = shuffle, basslines, voix, héritages house / jungle / R&B ;
pré-drop = tension armée, pas nécessairement résolution obligatoire ;
drop = événement rare, annulable, faussement résolutif ou déplacé.
```

## Traduction Vesperare

```text
pré-drop ne doit pas devenir recette ;
sub ne doit pas devenir spectaculaire par défaut ;
syncope peut servir tension / corps / relance ;
vide et retrait peuvent être fonctionnels ;
le système doit maintenir plusieurs sorties : résoudre, annuler, retarder, fausse résolution, retour au corps.
```

# Sources / cartes documentaires

## Source 1 — Dubstep

URL : https://en.wikipedia.org/wiki/Dubstep  
Type : synthèse historique.

### Ce que la source dit

Le dubstep émerge à South London au début des années 2000 comme offshoot UK garage, mêlant 2-step rhythms, sparse dub production, broken beat, grime, drum and bass. Il est généralement caractérisé par des patterns rythmiques syncopés, basslines proéminentes et une tonalité sombre.

### Notions Vesperare concernées

```text
pré-drop ;
tension syncopée ;
attente ;
sub ;
vide ;
sound system ;
résolution rare ;
pression ;
impact différé.
```

### Extraction Vesperare possible

```text
Le dubstep peut servir à penser l'attente, l'espace, la retenue et le sub comme masse.
Il ne doit pas imposer le drop.
La résolution peut être rare, annulée ou déplacée.
```

### Risques de mauvaise transposition

```text
confondre dubstep et brostep ;
importer un drop cliché ;
spectaculariser le sub ;
faire des effets identifiables non nécessaires ;
oublier la sparse production et le vide.
```

## Source 2 — 2-step garage

URL : https://en.wikipedia.org/wiki/2-step_garage  
Type : synthèse historique.

### Ce que la source dit

2-step est issu de la scène UK garage / jungle / pirate radio. Il est lié à Forward>>, à la continuité hardcore/jungle/garage, et à une évolution des patterns rythmiques.

### Notions Vesperare concernées

```text
syncope ;
déplacement du kick ;
shuffle ;
contretemps ;
relance ;
rythme non quatre-sur-le-floor ;
préparation dubstep.
```

### Extraction Vesperare possible

```text
La syncope peut servir à armer une tension sans drop.
Le déplacement ou retrait de certains appuis peut renforcer l'attente.
Le système peut utiliser la densité / retrait rythmique comme fonction, pas comme effet.
```

### Risques

```text
importer un groove UKG trop identifiable ;
confondre swing / shuffle / humanize ;
perdre le corps techno si la syncope devient trop décorative.
```

## Source 3 — UK garage

URL : https://en.wikipedia.org/wiki/UK_garage  
Type : synthèse historique.

### Ce que la source dit

Le passage speed garage → 2-step implique notamment la suppression du 2e et 4e kick dans la mesure, l'intérêt étant maintenu par des basslines syncopées et l'usage percussif d'autres instruments.

### Notions Vesperare concernées

```text
brillance rythmique ;
syncope ;
espace ;
percussion déplacée ;
sub/bassline ;
corps sans quatre-sur-le-floor constant.
```

### Extraction Vesperare possible

```text
Un groove peut être corporel même si certains appuis sont retirés.
La tension peut naître de ce qui manque, pas seulement de ce qui s'ajoute.
```

### Risques

```text
produire un style UKG reconnaissable ;
confondre retrait d'appui et perte de corps ;
utiliser la syncope comme ornament.
```

## Source 4 — Pitchfork, The Roots of Dubstep

URL : https://pitchfork.com/reviews/albums/9821-the-roots-of-dubstep  
Type : critique / contexte historique.

### Ce que la source suggère

Le dubstep early peut être pensé par sub, groove, echo, sculpture de basse, influences garage/dub-techno, et non par drop spectaculaire.

### Notions Vesperare concernées

```text
sub comme masse ;
echo / espace ;
groove sérieux ;
pré-drop ;
résolution ;
sound system ;
sculpture timbrale de basse.
```

### Extraction Vesperare possible

```text
Le sub peut être sculptural et spatial, pas seulement puissant.
La tension peut venir de l'espace et du retrait.
```

### Risques

```text
faire du dub-techno / dubstep identifiable ;
laisser l'écho devenir effet ;
confondre espace avec ambientisation.
```
