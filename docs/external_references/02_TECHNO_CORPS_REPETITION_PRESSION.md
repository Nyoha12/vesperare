# 02_TECHNO_CORPS_REPETITION_PRESSION

Version : v0.1  
Statut : références externes spécialisées. Non source de vérité Vesperare.

## Objet

Stocker les premiers résultats externes utiles pour réancrer la techno comme socle principal de Vesperare.

But : nourrir les notions techno-first sans réduire techno à kick / beat / sub / efficacité.

---

# Synthèse rapide pour extraction

## Apports utiles

```text
répétition = maintien actif du corps, pas simple boucle ;
minimalisme = réduction fonctionnelle, pas pauvreté ;
hypnose = stabilité + micro-mutation ;
pression = corps + spectre + densité + lisibilité ;
contour = attaques, relâchements, enveloppes et netteté du son ;
sub / low-end = fonction corporelle et orientation de masse, pas effet spectaculaire ;
techno = système de forces corporelles, temporelles, matérielles et formelles.
```

## Traduction Vesperare

```text
préserver fonction techno malgré naturalisation ;
réancrer impact, pression, hypnose, retour, répétition ;
concevoir les contrôles à partir de fonctions techno, pas de paramètres techniques ;
distinguer pression physique, spectrale, rythmique, densitaire et formelle ;
travailler micro-variations sans randomisation décorative.
```

# Sources / cartes documentaires

## Source 1 — Robert Hood / minimal techno / réduction / micro-variation

URL : https://pitchfork.com/reviews/albums/23380-paradygm-shift  
Type : critique musicale.

### Ce que la source dit

La critique de Robert Hood / Floorplan insiste sur des boucles arpégées, variations subtiles, micro-décalages et mutations entre rythmes imbriqués qui maintiennent une sensation de mouvement.

### Notions Vesperare concernées

```text
répétition non redondante ;
hypnose active ;
minimalisme fonctionnel ;
pression rythmique ;
variation subtile ;
corps techno ;
longue forme.
```

### Extraction Vesperare possible

```text
Répéter ne suffit pas : il faut maintenir une différence active.
La variation ne doit pas devenir décorative ou aléatoire visible.
La réduction techno peut renforcer le corps au lieu d'appauvrir la musique.
```

### Risques de mauvaise transposition

```text
confondre minimalisme et absence de matière ;
confondre micro-variation et randomisation ;
réduire Robert Hood à kick/sub ;
oublier la lisibilité corporelle.
```

Requêtes sources :


### Ce qu'il faut extraire


### Notions Vesperare concernées


### Extraction Vesperare possible

```text
Ne pas écrire pression = volume.
Ne pas écrire sub = puissance spectaculaire.
Définir pression comme combinaison de masse, densité, stabilité, lisibilité, confort, bas-médium, durée et attente.
```
