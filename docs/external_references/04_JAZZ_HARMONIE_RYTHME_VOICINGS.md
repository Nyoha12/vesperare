# 04_JAZZ_HARMONIE_RYTHME_VOICINGS

Version : v0.1  
Statut : références externes spécialisées. Non source de vérité Vesperare.

## Objet

Stocker les premiers résultats utiles pour réépaissir l'influence jazz sans la faire dominer Vesperare.

But : ne pas réduire le jazz à "couleur" ou "tension". Comprendre ce que cela peut apporter en voicings, placement rythmique, comping, rapport harmonie/rythme, tensions non résolues.

---

# Synthèse rapide pour extraction

## Apports utiles

```text
jazz = support / complément / réponse / tension / couleur / placement ;
comping = accompagnement actif, pas simple harmonie de fond ;
voicing = organisation concrète des tensions et couleurs ;
placement rythmique = rapport entre harmonie, accentuation et corps ;
tension jazz = souvent maintenue, déplacée, colorée ou suspendue, pas seulement résolue.
```

## Traduction Vesperare

```text
jazz comme couche de complexification, pas comme centre ;
utiliser champs harmoniques, couleurs et voicings distribués ;
relier jazz à partiels, microtonalité, gongs, didgeridoo, voix ;
penser finesse rythmique comme placement et accentuation, pas humanize ;
penser comping comme soutien actif d'un soliste/source live.
```

# Sources / cartes documentaires

## Source 1 — Comping (jazz)

URL : https://en.wikipedia.org/wiki/Comping_%28jazz%29  
Type : synthèse conceptuelle.

### Ce que la source dit

En jazz, comping désigne l'accompagnement — accords, rythmes, contre-mélodies — utilisé pour soutenir une ligne mélodique ou un solo improvisé.

### Notions Vesperare concernées

```text
accompagnement actif ;
réponse au live ;
soutien sans remplacement ;
voix / didgeridoo / guimbardes comme solistes ou pivots ;
rapport harmonie / rythme ;
finesse de placement ;
cohabitation système / instrument.
```

### Extraction Vesperare possible

```text
L'accompagnement système ne doit pas être une couche passive.
Il peut soutenir, compléter, contraster ou ponctuer la source live.
Le système peut jouer un rôle proche du comping : répondre sans prendre la place.
```

### Risques de mauvaise transposition

```text
importer des accords jazz reconnaissables ;
faire de l'harmonie décorative ;
confondre comping et remplissage ;
perdre le socle techno ;
transformer le système en accompagnateur trop bavard.
```
