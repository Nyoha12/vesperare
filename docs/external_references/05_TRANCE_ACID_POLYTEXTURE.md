Version : v0.1

Stocker les premiers résultats utiles sur trance, acid et polytexture.

But : contextualiser les notions de suspension, ouverture, expansion, peak non brutal, torsion résonante, ligne vivante, densité articulée et interlocking, sans importer les clichés de genre.

# Sources / cartes documentaires

## Source 1 — Acid techno

URL : https://en.wikipedia.org/wiki/Acid_techno  
Type : synthèse historique / stylistique.

### Ce que la source dit

L'acid techno dérive de l'acid house et applique le son squelching de la TB-303 à des matériaux techno plus durs. Le son acid est associé au filtre, à la résonance, au cutoff, aux accents, slides et octaves.

### Notions Vesperare concernées

```text
torsion résonante ;
ligne vivante ;
résonance contrôlée ;
mutation timbrale ;
hypnose par variation ;
acid naturalisé ;
formants ;
partiels ;
gorge / didgeridoo / guimbardes.
```

### Extraction Vesperare possible

```text
L'acid est utile comme principe de ligne timbrale mutante.
Il ne faut pas importer la TB-303 littéralement.
La torsion peut venir de formants, filtres, résonances, partiels, gorge, didgeridoo ou guimbardes.
```

### Risques de mauvaise transposition

```text
imitation de 303 ;
squelch décoratif ;
acid comme effet reconnaissable ;
perte de la matière acoustique ;
ligne trop autonome hors corps.
```

## Source 2 — Acid electronic music

URL : https://en.wikipedia.org/wiki/Acid_%28electronic_music%29  
Type : synthèse historique.

### Ce que la source dit

Acid désigne plusieurs styles électroniques employant le son squelching de la Roland TB-303. Le son acid est produit par résonance de filtre, cutoff, accent, slide et paramètres d'octave.

### Notions Vesperare concernées

```text
résonance ;
filtre ;
slide ;
accent ;
mutation ;
psychedelic quality ;
tension timbrale ;
ligne vivante.
```

### Extraction Vesperare possible

```text
Penser la torsion comme combinaison accent / glissement / résonance / mutation.
Relier acid à la modulation formantique, aux battements, aux micro-glissements et aux partiels.
```
