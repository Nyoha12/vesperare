# 00_INDEX_REFERENCES_EXTERNES

Version : v0.1  
Statut : dossier documentaire externe, non source de vérité Vesperare. À utiliser comme matériau de comparaison, contextualisation et extraction, sans remplacer les documents internes.

## Objet

Ce dossier stocke des références externes utiles pour Vesperare : recherches musicales, références méthodologiques, live électronique, NIME, techno, dubstep, jazz, trance, acid, polytexture et IA d'accompagnement.

Ces fiches ne sont pas des preuves finales et ne doivent pas créer directement :

```text
nouveau module ;
nouvelle règle ;
nouveau contrôle ;
nouveau mapping ;
nouveau prototype ;
nouveau source_need ;
seuil acoustique ;
choix esthétique définitif.
```

Elles servent à nourrir :

```text
analyse ;
comparaison ;
réancrage ;
questions ciblées ;
audit de solidité ;
révision des documents existants.
```

---

# Méthode d'utilisation

Pour chaque référence externe, toujours distinguer :

```text
fait ou description externe ;
interprétation possible pour Vesperare ;
risque de mauvaise transposition ;
document interne à relier ;
statut : utilisable / à vérifier / pure inspiration / danger.
```

Ne jamais importer un genre entier.

Ne jamais transformer une source web en règle Vesperare.

Toujours revenir aux documents internes :

```text
docs/05_CARTOGRAPHIE_STYLE_PERSONNEL_TECHNO.md
docs/01_CAHIER_DES_PHENOMENES_SONORES.md
docs/08_PERFORMANCE_SCENES_TRAJECTOIRES_OBJETS.md
docs/12_TECHNO_ORGANIQUE_CADRE_OPERATOIRE.md
docs/13_CONCEPTS_OPERATOIRES_TECHNO_ORGANIQUE_DEPENDANCES.md
docs/14_CONCEPT_CORE_TO_INTERFACE_SYSTEM_REQUIREMENTS.md
docs/23_SCENES_COMME_CHAMPS_OUVERTS_DE_PERFORMANCE.md

```
