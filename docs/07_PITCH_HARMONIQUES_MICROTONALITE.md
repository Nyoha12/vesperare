# 07_PITCH_HARMONIQUES_MICROTONALITE

Version : v0.1  
Statut : document de conception transversal.

## Objet

Ce document définit la manière dont Vesperare doit penser le pitch, les harmoniques, les partiels, les formants et la microtonalité.

Ce nœud traverse toutes les influences et toutes les familles de sons du projet :

```text
techno → corps, sub, centre grave ;
dubstep → sub, masse grave, tension / résolution ;
jazz → champs de tension, couleurs d’accords, voicings ;
trance → halos harmoniques, expansion, peak lumineux ;
acid → résonance, torsion, ligne vivante ;
Keita / polytexture → harmonie de timbres ;
gongs / bols / cloches → partiels inharmoniques ;
didgeridoo → fondamentale, formants, harmoniques ;
voix → hauteur, formants, cri, souffle, parole.
```

## Décision générale

La microtonalité doit être intégrée au projet, mais pas comme esthétique séparée.

Par défaut, elle doit agir comme :

```text
couleur ;
tension subtile ;
instabilité matérielle ;
battement ;
réalisme acoustique ;
expressivité de partiels ;
relation entre objets sonores.
```

Elle pourra devenir plus audible et assumée dans certains moments : suspensions, beautés autonomes, gongs exposés, champs harmoniques, transitions, pré-drops, lignes acid naturalisées ou peaks expansifs.

Formulation :

```text
La microtonalité est d’abord un outil de matérialité, de tension et de cohérence acoustique ;
elle peut devenir ponctuellement une dimension perceptible et structurante.
```

## 1. Le pitch n’est pas seulement une note

Dans Vesperare, il faut distinguer plusieurs régimes de hauteur.

### Hauteur-notée

Hauteur musicale classique : note, accord, mode, ligne mélodique.

Utile pour :

```text
certains accords reconnaissables ;
certains champs jazz ;
certains halos trance ;
certaines lignes acid naturalisées ;
certaines basses configurables ;
certaines voix ou cloches.
```

### Hauteur-ressentie

Un son donne une impression de hauteur sans être précisément accordé.

Exemples :

```text
grosse peau ;
gong grave ;
bol chantant ;
didgeridoo ;
voix criée ;
résonance métallique ;
sub texturé.
```

Le système doit pouvoir traiter ces sons comme ayant une direction tonale ou spectrale même s’ils ne correspondent pas parfaitement à une note.

### Centre de gravité spectral

Un son peut ne pas avoir de hauteur claire, mais posséder une zone d’énergie dominante.

Exemples :

```text
matière sombre ;
brillance aiguë ;
rugissement médium ;
souffle grave ;
frottement métallique.
```

Cette notion est importante pour organiser :

```text
grave / médium / aigu ;
corps / air / halo ;
pression / brillance / espace.
```

### Partiels

Les partiels sont les composants internes d’un son. Ils sont centraux pour :

```text
gongs ;
cloches ;
bols ;
cymbales ;
voix ;
didgeridoo ;
résonances naturelles ;
halos spectraux.
```

Un son peut être musicalement important non par une note, mais par son champ de partiels.

### Formants

Les formants sont des zones de résonance associées notamment à la voix, au didgeridoo, aux tubes, souffles ou corps résonants.

Ils peuvent agir comme des hauteurs expressives :

```text
voyelle ;
gorge ;
cri ;
rugissement ;
tube ;
didgeridoo ;
voix filtrée.
```

## 2. Pourquoi la microtonalité est nécessaire

La microtonalité devient nécessaire parce que beaucoup de sources du projet ne sont pas tempérées :

```text
gongs ;
bols ;
cloches ;
didgeridoo ;
voix ;
cris ;
résonances naturelles ;
field recordings ;
objets acoustiques ;
partiels inharmoniques.
```

Forcer ces sources dans une grille tempérée standard ferait perdre :

```text
richesse matérielle ;
battements ;
instabilité expressive ;
tensions naturelles ;
halos complexes ;
réalisme acoustique.
```

Mais laisser tout libre sans organisation ferait courir le risque inverse :

```text
flou harmonique ;
boue spectrale ;
absence de centre ;
incompatibilité avec basses, drops ou peaks ;
contrôle live trop difficile.
```

Donc Vesperare doit viser une microtonalité organisée.

## 3. Fonctions de la microtonalité

### Tension

Des hauteurs proches créent battements, friction, attente et mystère.

Exemples :

```text
deux partiels proches ;
deux voix légèrement décalées ;
un gong + un sub ;
un bol + une cloche.
```

Utile pour :

```text
dubstep attente ;
jazz champ de tension ;
acid torsion ;
trance suspension.
```

### Résolution

Une microtension peut se stabiliser.

Exemples :

```text
battement → unisson approximatif ;
friction → centre ;
hauteur instable → grave stable ;
halo tendu → champ clair.
```

Utile pour :

```text
drop ;
peak sans brutalité ;
retour au corps ;
résolution de gong ;
stabilisation du sub.
```

### Naturalisation

La microtonalité permet de conserver la sensation d’objets réels et de phénomènes physiques. Elle empêche les sons d’être trop abstraitement MIDI ou synthétiques.

### Expressivité vocale / corporelle

Les voix, cris, souffles, didgeridoo, glissements de formants ne sont pas naturellement fixes. La microtonalité peut donner :

```text
humanité ;
tension ;
corps ;
fragilité ;
puissance ;
animalité.
```

### Harmonie élargie

Le jazz apporte les couleurs d’accords ; la microtonalité peut les enrichir par battements, voicings non tempérés, résonances et partiels.

## 4. Harmoniques comme pont entre influences

### Dubstep

Le sub n’est pas seulement une fondamentale. Il peut avoir des harmoniques, un rugissement, une voix, un gong, une saturation douce ou des battements microtonaux.

Le drop peut donc être aussi une résolution harmonique ou spectrale.

### Jazz

Les couleurs d’accords peuvent devenir des relations entre partiels distribués entre sources : grave, gong, voix, cloche, halo.

### Trance

L’expansion peut être une ouverture des harmoniques : plus de partiels aigus, plus de clarté, plus de résonance large, plus d’accords suspendus.

### Acid

La torsion peut être une déformation des harmoniques : filtre, formants, résonance, partiels révélés ou resserrés.

### Polytexture percussive

L’harmonie de percussions vient aussi de la complémentarité de timbres et de partiels : bois, peau, métal, grave court, halo léger.

## 5. Régimes d’organisation de hauteur

Le système devra permettre plusieurs régimes, pas comme modes stylistiques séparés, mais comme régimes d’organisation sonore.

### Régime tempéré

Utile pour accords reconnaissables, certains éléments trance, certaines progressions, certaines lignes mélodiques, compatibilité Ableton / MIDI.

### Régime spectral

Les hauteurs dérivent des partiels d’un son source : gong, bol, didgeridoo, voix, cloche, sub.

Très compatible avec Vesperare.

### Régime microtonal contrôlé

Les hauteurs peuvent s’écarter du tempérament, mais avec limites.

Utile pour tension, battement, instabilité, acid naturalisé, jazz spectral, trance suspendue.

### Régime modal

Un mode ou une collection de hauteurs stabilise une scène.

Peut venir de :

```text
mode inventé ;
partiels sélectionnés ;
couleur jazz ;
instrument acoustique ;
drone ;
field recording analysé.
```

### Régime bruité / non-pitché

Certaines couches n’ont pas de pitch clair : percussions, brillance rythmique, frottements, air, sons naturels.

Même là, le centre spectral doit être contrôlé.

## 6. Ce que le système devra savoir faire

### Estimer ou déclarer le centre d’un son

Pour chaque matière sonore :

```text
fondamentale claire ?
hauteur ressentie ?
zone spectrale dominante ?
partiels importants ?
inharmonicité ?
```

### Relier les sons par familles de partiels

Exemple :

```text
un gong fournit un champ de partiels ;
les cloches choisies renforcent certains partiels ;
la basse se place sur une fondamentale compatible ;
le halo utilise une couleur dérivée.
```

### Permettre des écarts microtonaux contrôlés

```text
écart faible → couleur / battement ;
écart moyen → tension ;
écart fort → étrangeté / instabilité.
```

### Choisir le degré de stabilité harmonique

```text
stable ;
tendu ;
flottant ;
instable ;
résolu ;
suspendu.
```

### Lier harmonie et forme

Exemples :

```text
pré-drop → microtension grave ;
drop → stabilisation du centre ;
suspension trance → tension suspendue ;
peak → rayonnement harmonique ;
acid → torsion de partiels ;
jazz → champ d’accords élargi.
```

## 7. Risques

### Complexité ingérable

Microtonalité, partiels, accords, modes, gongs, voix, sub, acid, trance et jazz peuvent produire un système trop lourd. Il faudra organiser par niveaux de contrôle.

### Perte du corps

Trop de sophistication harmonique peut affaiblir l’impact, le sub, la pulsation et la fonction corporelle.

### Flou harmonique

Si tous les éléments ont des partiels riches et instables, le résultat peut devenir boueux, fatigant, sans centre.

### Trop de beauté autonome

Les gongs, bols, halos et voix peuvent devenir fascinants mais décoratifs. Il faudra relier la beauté à la forme, même lorsqu’elle est autonome.

## 8. Implications Max for Live

Un futur sous-système pourra être nécessaire :

```text
Pitch / Harmonic Field Manager
```

Rôle possible : coordonner centres de hauteur, échelles, partiels, microtonalité, accords / champs, halos, sub, voix, gongs, résonateurs.

Ce module ne devrait pas être trop visible en live. Il devrait agir principalement comme système interne, avec quelques contrôles simples.

Variables possibles :

```text
pitch_center
spectral_root
microtonal_spread
harmonic_tension
partial_alignment
field_stability
inharmonicity_amount
resolution_target
sub_pitch_relation
halo_tuning
```

Contrôles live possibles :

```text
Stabilité harmonique ;
Tension microtonale ;
Ouverture harmonique ;
Centre grave ;
Alignement des partiels ;
Suspension ;
Résolution.
```

## 9. Décision actuelle

La microtonalité doit donc être conçue selon deux régimes complémentaires :

```text
1. régime par défaut : couleur, tension subtile, battements, matérialité ;
2. régime ponctuel : microtonalité perceptible, assumée, structurante, dans certains moments de forme.
```

Elle doit toujours rester reliée à :

```text
la fonction corporelle ;
les partiels des sources ;
les gongs / bols / cloches / voix / didgeridoo ;
les halos ;
les basses ;
les tensions jazz ;
les suspensions trance ;
les drops dubstep ;
les torsions acid naturalisées.
```

## Formule directrice

Dans Vesperare, le pitch doit être pensé comme une organisation de forces spectrales, corporelles, harmoniques et microtonales, plutôt que comme une simple grille de notes MIDI.
