# 01_CAHIER_DES_PHENOMENES_SONORES

Version : v0.2  
Statut : cahier des phénomènes sonores du projet.

## Objet

Ce document définit les phénomènes sonores centraux de Vesperare. Il sert de pont entre la cartographie du style personnel techno et la future spécification Max for Live.

Le projet est compris comme une techno personnelle naturalisée, où les fonctions techno sont conservées par leurs effets perceptifs et corporels, mais portées par des matières réelles, acoustiques, instrumentales, vocales, naturelles ou synthétisées comme physiques.

## Principe général

Une fonction techno peut être conservée même si son support sonore change.

On ne demande pas seulement :

```text
ce son est-il techno ?
```

mais :

```text
quelle fonction techno ce son peut-il porter ?
est-ce qu’il la porte avec assez de force ?
qu’ajoute sa matière naturelle, acoustique ou instrumentale ?
qu’est-ce qu’il risque de faire perdre ?
```

## 1. Impact-fondation corporel

### Définition

Phénomène par lequel un son réinstalle le corps de la musique : sol, pulsation, impact, autorité temporelle, pression physique.

Il correspond à une fonction souvent portée par le kick techno, mais il ne doit pas nécessairement sonner comme un kick électronique.

### Supports possibles

```text
tambour profond ;
peau grave ;
grosse percussion traitée ;
impact acoustique ;
objet frappé ;
gong très sculpté ;
didgeridoo percussif ou filtré ;
résonance grave resserrée ;
son naturel grave transformé ;
synthèse à comportement acoustique.
```

### Ce qui la rend désirable

Elle permet de garder la force techno même lorsque les sons deviennent naturels ou instrumentaux. Elle donne à l’auditeur une certitude corporelle.

### Risques

```text
trop de résonance → fonction devenue halo ;
grave flou → perte d’autorité ;
attaque molle → perte de danse ;
irrégularité excessive → perte d’ancrage ;
timbre illustratif → perte de tension techno.
```

### Variables possibles

```text
impact_weight
low_authority
body_pressure
tail_control
sub_stability
foundation_regularity
```

## 2. Pression spectrale corporelle

### Définition

Sensation que le son occupe physiquement et mentalement l’espace d’écoute. Elle ne dépend pas seulement du volume, mais de la qualité du grave, du bas-médium, de la densité, de la stabilité, du contour et du confort spectral.

### Fonction dans le projet

Garder l’intensité techno et hardcore sans utiliser principalement l’agression des aigus, le volume excessif ou la dureté punitive.

### Supports possibles

```text
sub contrôlé ;
didgeridoo filtré ;
drone grave organique ;
gongs filtrés ;
peaux graves ;
résonances compactées ;
matières naturelles ralenties ou resserrées ;
basse synthétisée comme phénomène acoustique.
```

### Risques

```text
boue bas-médium ;
sub instable ;
pression qui écrase les détails ;
fatigue auditive ;
confusion entre intensité et volume.
```

### Variables possibles

```text
spectral_pressure
sub_stability
mid_pressure
body_intensity
comfort_limit
low_density
```

## 3. Contour sculpté / netteté sensible

### Définition

Remplacement de l’idée trop froide de “sécheresse”. Le projet ne cherche pas nécessairement une techno froide, sèche ou aride. Ce qui est retenu : son dessiné, attaque lisible, queue maîtrisée, matière bien formée, enveloppe raffinée, son fini et bien senti.

### Critère

Le critère n’est pas :

```text
est-ce sec ?
```

mais :

```text
est-ce que le son tient sa fonction sans se dissoudre ?
est-ce que son attaque et sa queue sont maîtrisées ?
est-ce que sa richesse reste lisible ?
```

### Supports possibles

```text
peaux courtes ;
bois frappés ;
petits métaux mats ;
cymbalettes contrôlées ;
cloches courtes ;
voix coupées ;
souffles tranchés ;
batterie resculptée ;
objets percussifs ;
sons naturels édités finement.
```

### Variables possibles

```text
attack_definition
tail_control
envelope_clarity
grain_presence
function_readability
halo_permission
```

## 4. Hypnose active

### Définition

Répétition qui porte, aspire et entraîne. Elle n’est pas passive ou ambient. Elle garde un rapport au corps, au retour, à la tension et à la propulsion.

### Caractéristiques

```text
cadre stable ;
retour perceptible ;
motif fort ;
micro-variation signifiante ;
pression corporelle ;
détail rythmique valorisé ;
mutation lente.
```

### Risques

```text
trop de variation → perte d’hypnose ;
trop de texture → ambientisation ;
trop de densité → saturation ;
trop peu de corps → décor sonore ;
répétition morte → redondance.
```

### Variables possibles

```text
hypnotic_stability
loop_identity
mutation_rate
body_continuity
anchor_strength
micro_variation_depth
```

## 5. Intensité phénoménale sans agression

### Définition

Intensité forte, directe, presque hardcore, qui ne repose pas principalement sur violence fréquentielle ou volume.

Elle vient plutôt de :

```text
singularité du son ;
pression maîtrisée ;
impact ;
surprise ;
sur-réalité ;
occupation totale de l’espace ;
densité physique ;
qualité du phénomène acoustique.
```

### Risques

```text
harshness ;
aigus agressifs ;
volume punitif ;
saturation non maîtrisée ;
fatigue ;
perte de détail.
```

### Variables possibles

```text
phenomenal_intensity
comfort_guard
harshness_limit
transient_power
spectral_density
impact_without_loudness
```

## 6. Brillance rythmique naturalisée

### Définition

Lisibilité temporelle et propulsion souvent portées par hats, rides et micro-percussions techno, mais pouvant être portées ici par sons aigus naturels, instrumentaux ou naturalisés.

### Supports possibles

```text
insectes ;
oiseaux ;
froissements ;
grains ;
cliquetis ;
petites cloches ;
clochettes ;
cymbalettes ;
cymbales fines ;
petits métaux ;
souffles courts ;
shakers naturels ;
micro-objets.
```

### Risques

```text
aigus fatigants ;
bruit blanc organique ;
décor naturaliste ;
ornementation sans propulsion ;
texture permanente.
```

### Variables possibles

```text
air_density
brightness_comfort
micro_pulse_strength
natural_irregularity
time_readability
high_transient_control
```

## 7. Polytexture percussive raffinée

### Définition

Tissage de percussions, timbres et accents qui crée une complexité lisible et harmonieuse. Liée à l’inspiration Keita Ogawa, mais pas comme style importé.

### Apports

```text
interlocking ;
groove composite ;
harmonie de timbres percussifs ;
complexité lisible ;
densité articulée ;
richesse acoustique ;
contrepoint rythmique.
```

### Risques

```text
virtuosité décorative ;
perte du cadre techno ;
médiums trop chargés ;
style importé involontaire ;
confusion des accents ;
trop de couches équivalentes.
```

### Variables possibles

```text
interlock_amount
percussive_density
accent_hierarchy
mid_load
timbre_spacing
groove_composite_strength
```

## 8. Espace musical occupé

### Définition

Manière dont les sons créent profondeur, présence, scène et immersion. Ce n’est pas simplement reverb ou delay.

### Fonctions

```text
nuit ;
mystère ;
distance ;
présence ;
intensité ;
incertitude ;
profondeur.
```

### Supports possibles

```text
résonances de gongs ;
bols chantants ;
cloches ;
espaces enregistrés ;
halos spectraux ;
delays filtrés ;
réverbérations courtes ;
voix spatialisée ;
objets dans des lieux ;
queues naturelles maîtrisées.
```

### Risques

```text
ambientisation ;
halo permanent ;
perte d’impact ;
flou rythmique ;
espace décoratif ;
perte de proximité.
```

### Variables possibles

```text
space_depth

```

## 9. Voix comme matière, instrument ou signal

### Définition

La voix peut être présente sans devenir chanson. Elle peut être instrument, effet, matière, fragment, signal, texture, présence humaine, hook minimal ou événement formel.

### Possibilités

```text
voix parlée ;
fragment de phrase ;
souffle ;
mot isolé ;
voix rythmée ;
voix coupée ;
voix filtrée ;
voix brute ;
voix texturale ;
parole semi-compréhensible ;
hook minimal.
```

### Risques

```text
basculer vers la chanson ;
prendre tout le centre ;
devenir gimmick ;
sur-signifier ;
affaiblir l’énergie techno.
```

### Variables possibles

```text
voice_presence
semantic_clarity
voice_rhythm
voice_texture
voice_priority
vocal_halo
```

## 10. Beauté autonome / moment de matière

### Définition

Moments où un son existe pour sa beauté propre, sa présence matérielle ou sa valeur sensible, même s’il ne remplit pas immédiatement une fonction techno stricte.

### Supports possibles

```text
bol chantant ;
gong long ;
résonance naturelle ;
voix nue ;
son pur ;
objet acoustique isolé ;
matière de nature ;
instrument exposé.
```

### Risques

```text
perte de tension ;
parenthèse décorative ;
ambientisation ;
affaiblissement du style techno ;
moment trop long ou trop détaché.
```

### Variables possibles

```text
autonomous_beauty_permission
form_suspension
material_focus
techno_distance
return_to_body
```

## Relation avec les phénomènes plus anciens

Les notions de densité, grain, halo, saillance, équilibre spectral, partiels, masquage, glissement d’appui et interaction live restent pertinentes. Elles doivent maintenant être relues à travers ce projet de techno personnelle naturalisée.

Exemples :

```text
densité → pression corporelle + lisibilité techno ;
grain → contour sculpté + matière acoustique ;
halo → espace musical + risque d’ambientisation ;
saillance → impact-fondation + voix + événement formel ;
spectre → confort, pression, intensité sans agression ;
polytexture → enrichissement techno, pas importation de style.
```

## Question directrice

Le phénomène conserve-t-il sa force techno tout en gagnant la singularité matérielle, organique, acoustique, spectrale, instrumentale ou vocale recherchée par le projet ?
