# SOURCE_INGESTION_CHECKLIST

## Purpose

Use this checklist when turning raw material into processed knowledge.

Raw material is not knowledge until processed.

## Checklist

- [ ] Raw source is preserved or referenced.
- [ ] Source title is identified.
- [ ] Source type is identified.
- [ ] Date is included if known.
- [ ] Context of use is explained.
- [ ] Summary is separated from interpretation.
- [ ] Key ideas are extracted.
- [ ] Claims are listed.
- [ ] Uncertainty is explicit.
- [ ] Possible applications are listed.
- [ ] Verification needs are listed.
- [ ] Quotes, paraphrases, and inferences are separated.
- [ ] The processed note links back to the raw source.

## Failure conditions

Reject ingestion if:

- it summarizes without provenance;
- it treats claims as facts;
- it loses uncertainty;
- it flattens subtle material into generic advice;
- it does not preserve the raw source.

## Output

PASS / NEEDS REVISION / REJECT
