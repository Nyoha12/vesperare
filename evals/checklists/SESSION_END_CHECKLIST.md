# SESSION_END_CHECKLIST

## Purpose

Use this checklist at the end of any substantial AI-assisted session.

## Checklist

- [ ] The session goal is clear.
- [ ] Files read are listed.
- [ ] Facts from files are separated from inference.
- [ ] User intention is separated from model recommendation.
- [ ] Decisions are listed.
- [ ] Uncertainties are listed.
- [ ] Verification needs are listed.
- [ ] Any changed files are listed.
- [ ] Repeated mistakes or corrections are captured.
- [ ] Potential skill updates are identified.
- [ ] Next minimal action is defined.
- [ ] Things not to do next are defined.

## Output

```md
# SESSION_SUMMARY

## Goal

## Files read

## Established

## Uncertain

## Decisions

## Verification needed

## Changed files

## Durable learnings

## Skill updates needed

## Next minimal action

## Do not do next
```
